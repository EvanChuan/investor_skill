# 行為金融學原則應用指南
# Behavioral Finance Principles for Market Sentiment Analysis

## 概述

行為金融學研究投資人在不確定環境下的非理性決策模式，是情緒追蹤框架的理論基礎。本文件整理投資實務中最具解釋力的行為偏誤，並說明如何在市場情緒分析中識別與應用這些模式。

---

## 一、核心行為偏誤分類

### 1.1 認知偏誤（Cognitive Biases）

#### 確認偏誤（Confirmation Bias）
- **定義：** 投資人傾向尋找並接受支持自己既有觀點的資訊，忽略反面證據
- **市場表現：** 多頭行情中，投資人只看好消息；空頭市場中，只關注壞消息
- **識別信號：**
  - 社群媒體一面倒的情緒（看多比例 > 85% 或 < 15%）
  - 分析師評級持續向同一方向移動（連續升評或降評超過 5 次/週）
- **應用：** 確認偏誤嚴重時，情緒極端值更具反向投資意義

#### 錨定效應（Anchoring Bias）
- **定義：** 投資人過度依賴最初取得的參考資訊（錨點）做判斷
- **市場表現：** 股票從高點回落 20%，投資人認為「便宜了」，卻忽略基本面已惡化
- **識別信號：**
  - 分析師目標價長期未調整，遠偏離當前股價
  - 市場反覆討論「歷史高點/低點」作為支撐壓力參考
- **應用：** 錨定效應可能導致估值錯誤，需交叉驗證當前基本面

#### 可得性偏誤（Availability Bias）
- **定義：** 投資人過度重視容易回想起來的事件（通常是近期發生的）
- **市場表現：** 近期發生崩盤後，過度低估未來報酬；近期大漲後，過度高估未來報酬
- **識別信號：**
  - 媒體密集報導某一類風險後，相關 ETF 資金大量撤離
  - 近期 IPO 熱潮或某類股表現強勢後，新開戶數激增
- **應用：** 識別「過度反應」的本質，評估情緒是否超越理性評估

#### 過度自信（Overconfidence Bias）
- **定義：** 投資人高估自己的判斷能力與預測準確度
- **市場表現：** 牛市中期出現槓桿使用量暴增、集中押注單一標的
- **識別信號：**
  - 融資餘額快速攀升（散戶加槓桿）
  - 選擇權 Call 買盤量大幅超過 Put（Put/Call Ratio < 0.5）
  - 散戶持倉集中度創高
- **應用：** 過度自信通常是多頭末期信號，搭配估值過高則為強烈警訊

### 1.2 情緒偏誤（Emotional Biases）

#### 損失規避（Loss Aversion）
- **定義：** 損失帶來的痛苦約為同等獲利帶來的愉悅感 2 倍（Kahneman & Tversky）
- **市場表現：**
  - 投資人不願意認賠，持有虧損部位過長（處置效應）
  - 恐慌性拋售時，殺跌力道往往超越理性估值所能解釋的幅度
- **識別信號：**
  - 融資斷頭賣壓（股價跌破維持率觸發強制賣出）
  - VIX 短時間內急升 > 30%（恐慌性拋售）
  - 成交量異常放大同時股價急跌（非理性拋售）
- **應用：** 損失規避導致的恐慌性拋售是最佳逆向進場機會之一，但需確認基本面完整

#### 處置效應（Disposition Effect）
- **定義：** 投資人傾向過早賣出獲利部位、過晚賣出虧損部位
- **市場表現：** 股價創新高後成交量放大（部分投資人獲利了結），造成阻力
- **識別信號：**
  - 成交量密集的歷史套牢區域（技術面支撐壓力的行為金融學解釋）
  - 短線獲利了結導致漲勢停滯，但基本面仍佳
- **應用：** 識別套牢成本區作為關鍵支撐，評估解套賣壓對上漲的影響

#### 羊群效應（Herding Behavior）
- **定義：** 投資人傾向追隨多數人的行為，而非依據自身分析
- **市場表現：** 趨勢加速、泡沫形成、崩潰加速
- **識別信號：**
  - 社群討論量暴增（> 平均值 5 倍）且方向一致
  - ETF 資金流與個別股票資金流方向高度一致
  - 機構持倉高度集中在少數熱門標的（13F 數據）
  - 新開戶數與市場高點同步出現
- **應用：** 羊群效應在末段最明顯，配合估值過熱可識別泡沫頂部

#### 後悔規避（Regret Aversion）
- **定義：** 投資人為了避免「後悔感」而做出次優決策
- **市場表現：**
  - 不敢在恐慌中買進（怕買進後繼續跌）
  - 不敢在高位減碼（怕賣出後繼續漲而後悔）
- **識別信號：**
  - 極度恐慌時，即使低估值標的也無人問津
  - 極度貪婪時，理性投資人遲遲不願減碼
- **應用：** 後悔規避導致「應該做但沒做」，是情緒極端後反轉的動力來源

### 1.3 社會偏誤（Social Biases）

#### FOMO（Fear of Missing Out，害怕錯過）
- **定義：** 因擔心錯過上漲機會而倉促追高
- **市場表現：** 市場快速上漲時，散戶大量湧入、成交量放大
- **識別信號：**
  - 零股交易量暴增（散戶小額追買）
  - Reddit / PTT / StockTwits 討論量急升
  - 新開戶數創高點
  - Fear & Greed Index 快速從中性躍升至貪婪區（單週 +20 分以上）
- **應用：** FOMO 通常出現在多頭中後期，是減碼的警訊而非加碼信號

#### 社會認同（Social Proof）
- **定義：** 在不確定情況下，參考他人行為來決策
- **市場表現：** 名人/KOL 推薦股票後引發跟風
- **識別信號：**
  - 特定股票被知名投資人買進的消息後，散戶跟進
  - 13F 公告後出現跟單行為（機構持倉公告後相關股票短期上漲）
- **應用：** 識別「跟單效應」，評估背後是否有基本面支撐

## 二、行為偏誤與情緒評分的對應關係

| 行為偏誤 | 觸發的市場情緒 | 對應情緒評分影響 | 逆向機會 |
|---------|-------------|----------------|---------|
| 損失規避 + 恐慌 | 極度恐慌 (0-20) | VIX 飆升、Put/Call 激增 | 高（基本面未惡化時） |
| FOMO + 羊群效應 | 極度貪婪 (80-100) | 散戶狂熱、融資暴增 | 賣出信號 |
| 確認偏誤 | 情緒固化（單向） | 社群 / 媒體一致 | 反轉預兆 |
| 過度自信 | 多頭中後期貪婪 | 槓桿增加、Call 大量買入 | 注意反轉 |
| 後悔規避 | 極端情緒後的僵局 | 量縮整理 | 等待觸媒 |
| 處置效應 | 創新高後的阻力 | 成交量放大但漲勢停滯 | 整理後再攻 |

## 三、行為金融學在逆向投資中的應用

### 3.1 識別「非理性極端」的四步驟

```
Step 1: 確認情緒指標到達極端值
  → 綜合情緒評分 < 20 或 > 80

Step 2: 識別主導行為偏誤
  → 是損失規避（恐慌）還是 FOMO（貪婪）？
  → 是羊群效應（方向一致）還是確認偏誤（資訊選擇性）？

Step 3: 評估基本面是否支撐逆向操作
  → 極端恐慌 + 基本面完整 → 逆向買進機會
  → 極端貪婪 + 估值過高 → 逆向賣出機會

Step 4: 等待「行為反轉觸媒」
  → 情緒分數不再創新極端值（連續 3 天）
  → 成交量變化（恐慌：量縮；貪婪：量萎縮）
  → 媒體敘事開始轉向
```

### 3.2 常見的反轉模式

**恐慌型反轉（底部訊號）：**
1. VIX 急升後開始回落
2. Put/Call Ratio 從高點下降
3. 負面新聞比例雖高，但股價已停止下跌
4. 內部人（Insider）開始買進
5. 機構法人由賣超轉為小額買超

**貪婪型反轉（頂部訊號）：**
1. Fear & Greed Index 維持 80+ 超過 2 週後開始回落
2. 融資增速放緩（新進散戶減少）
3. 利好消息無法帶動股價繼續上漲（利多出盡）
4. 機構法人開始低調減碼
5. 媒體從「理性分析」轉為「封面效應」（財經雜誌頭版）

## 四、封面效應（Magazine Cover Indicator）

**原則：** 當某個投資主題登上主流媒體封面時，往往預示趨勢即將反轉。

**解讀：**
- 正面封面（「科技股還能漲多少？」「AI 改變世界」）→ 頂部訊號
- 負面封面（「股市崩潰了？」「你的退休金不見了」）→ 底部訊號

**應用方法：**
- 搜索近 1 個月財經媒體（Bloomberg、BusinessWeek、Forbes、Money）封面主題
- 與當前情緒評分交叉驗證
- 封面效應通常領先反轉 1-4 週

## 五、行為偏誤快速識別表

| 市場現象 | 可能的行為偏誤 | 建議行動 |
|---------|-------------|---------|
| 散戶大量湧入、新開戶創高 | FOMO + 羊群效應 | 謹慎追高，考慮減碼 |
| VIX 急升，成交量爆量下跌 | 損失規避 + 恐慌 | 評估基本面，尋找逆向機會 |
| 社群 90% 以上看多，但股價不再創高 | 確認偏誤 + 過度自信 | 頂背離警訊，準備減碼 |
| 媒體一致負面，但量縮股價止跌 | 損失規避 + 可得性偏誤 | 底部訊號，分批試單 |
| 分析師快速集體升評 | 羊群效應 + 後見之明 | 已反映，追高需謹慎 |
| 知名投資人持倉曝光後跟單 | 社會認同 | 確認基本面再決策 |
| 融資暴增但股價漲幅縮小 | 過度自信 + 處置效應 | 籌碼惡化，上漲空間有限 |
