# 跨資產資金流追蹤分析 (Cross-Asset Fund Flow Tracker)

## 概述

本模組是第一階段「總體經濟分析」的核心組成部分，提供**完全自動化的跨資產資金流分析系統**。透過整合三大數據來源，幫助投資者在進行個股選擇之前，先判斷「資金正往哪些資產類別流動」。

### 核心理念

**資產配置決定 80% 的報酬**

- 在資金大規模撤出股票時，再好的個股也難逃下跌
- 在資金湧入債券時，持有股票的機會成本極高
- 瞭解資金流向，才能做對大方向的配置決策

### 三重驗證機制

本模組整合三大數據來源，提供高可靠度的資金流判斷：

```
【數據來源 1】ETF / 基金資金流
↓ 反映：現貨市場的實際買賣行為
↓ 優勢：即時性高、代表散戶與機構的實際動作

【數據來源 2】CFTC COT 大戶持倉
↓ 反映：期貨市場專業投資者的真實部位
↓ 優勢：領先指標、揭露大戶真實意圖

【數據來源 3】機構資產配置觀點
↓ 反映：PIMCO、JP Morgan、BlackRock 等的戰略建議
↓ 優勢：長期視角、戰略配置方向

三重驗證
↓ 當三者一致時：高信度資金流信號
↓ 當三者矛盾時：需觀望或降低部位
```

### 步驟 2：查詢各 ETF 的「資金流」與「價格動能」

#### 2.1 ETF 資金流（Fund Flows）

**目標**：判斷「資金是否真的在流入這個資產」

**查詢來源（按優先順序）**：

1. **ETFdb.com**
   - 搜尋關鍵字：`[ETF ticker] fund flows`（例如 `QQQ fund flows`）
   - 關注欄位：`1-Week Flow`、`1-Month Flow`、`3-Month Flow`

2. **Morningstar Fund Flows**
   - 搜尋：`Morningstar [ETF ticker] fund flows`
   - 可作為 ETFdb 的補充或交叉驗證

3. **ETF 發行商官網**（iShares、Vanguard、State Street）
   - 查看「Assets Under Management (AUM)」的變化
   - 比較近期與前期 AUM，估算資金流

**判斷標準**：

| 資金流規模 | 標記 | 分數 |
|---|---|---|
| 淨流入 > $1B/週 | 大幅淨流入 | +2 |
| 淨流入 $0–$1B | 淨流入 | +1 |
| -$200M ~ +$200M | 中立 | 0 |
| 淨流出 $0–-$1B | 淨流出 | -1 |
| 淨流出 < -$1B/週 | 大幅淨流出 | -2 |

**輸出格式（每個 ETF）**：

```
ETF: QQQ
近 1 週資金流：淨流出 $-800M (-1)
近 1 個月資金流：淨流出 $-2.5B
```

---

#### 2.2 價格動能（Price Momentum）

**目標**：判斷「價格走勢是否與資金流一致」

**查詢來源**：
- Yahoo Finance / Google Finance / TradingView
- 搜尋：`[ETF ticker] stock price`

**關注指標**：
- 近 1 週漲跌幅（%）
- 近 1 個月漲跌幅（%）
- 近 3 個月漲跌幅（%）

**判斷標準**：

| 漲跌幅表現 | 標記 | 分數 |
|---|---|---|
| 近 1 週 +2% 且近 1 月 +5% 且近 3 月 +10% | 強勢上漲 | +2 |
| 近期多數時間上漲 | 上漲 | +1 |
| 震盪區間（±2% 內） | 盤整 | 0 |
| 近期多數時間下跌 | 下跌 | -1 |
| 近 1 週 -2% 且近 1 月 -5% 且近 3 月 -10% | 大幅下跌 | -2 |

**輸出格式**：

```
ETF: QQQ
近 1 週：-1.2%
近 1 個月：-3.5%
近 3 個月：+8.2%
價格動能分數：-1（下跌）
```

---

#### 2.3 成交量變化（可選，驗證買盤強度）

**目標**：確認「價格上漲是否伴隨成交量放大」

**查詢來源**：
- Yahoo Finance、TradingView 的「Volume」欄位
- 比較近 1 週平均成交量 vs 過去 3 個月平均成交量

**判斷標準**：

| 成交量變化 | 標記 | 意義 |
|---|---|---|
| 放大 > 20% | 有量配合 | 真實買盤 |
| -20% ~ +20% | 量能正常 | 中性 |
| 萎縮 > 20% | 縮量 | 可能是假突破 |

---

### 步驟 3：查詢 CFTC COT 報告（大戶持倉分析）

**目標**：判斷「大型投機者」（避險基金、CTA）的期貨持倉方向與變化

#### 3.1 CFTC COT 報告簡介

**Commitments of Traders (COT)** 是 CFTC 每週發布的期貨持倉報告，揭露三類市場參與者的部位：

| 類別 | 英文 | 特性 | 對市場的影響 |
|---|---|---|---|
| **商業避險者** | Commercial Hedgers | 實體企業（礦商、石油公司），做反向避險 | 逆勢指標（低點做空、高點做多） |
| **大型投機者** | Non-Commercial | 避險基金、CTA、大型機構 | **同向指標**（做多 → 價格漲） |
| **小型投機者** | Non-Reportable | 散戶 | 反向指標（追高殺低） |

> **重點關注「大型投機者（Non-Commercial）」的「淨多單（Net Long）」變化。**

---

#### 3.2 查詢來源

1. **CFTC 官方網站**
   - 網址：https://www.cftc.gov/MarketReports/CommitmentsofTraders/index.htm
   - 發布時間：每週五下午 3:30（統計截至週二收盤）

2. **第三方視覺化工具（推薦）**
   - **Barchart COT**：https://www.barchart.com/futures/commitment-of-traders
   - **Finviz COT Charts**：https://finviz.com/futures_charts.ashx
   - **COT Base**：https://www.cotbase.com/

---

#### 3.3 判斷標準與評分

**查詢「大型投機者淨多單」的近期變化（至少看 2–3 週）**：

| COT 淨多單變化 | 信號 | 分數 | 意義 |
|---|---|---|---|
| 連續 2 週以上增加 | ✅ 強烈做多 | +0.5 | 大戶持續加碼，資金流入確認 |
| 變化 < ±5% | ⚠️ 中性/觀望 | 0 | 大戶觀望，或多空均衡 |
| 連續 2 週以上減少 | ❌ 強烈做空 | -0.5 | 大戶減倉或做空，資金流出警訊 |

**輸出格式（每個資產）**：

```
資產：黃金 (GLD)
COT 合約：GC (COMEX Gold)
大型投機者淨多單：+285,000 口
週變化：+15,000 口（連續 3 週增加）
COT 信號：✅ 強烈做多 (+0.5)
與 ETF 資金流一致性：✅ 一致（GLD 淨流入 $900M）
```

---

#### 3.4 散戶 vs 大戶對比（逆向指標驗證）

**額外查看「小型投機者」部位，作為反向驗證**：

| 大型投機者 | 小型投機者 | 判斷 |
|---|---|---|
| ✅ 做多 | ❌ 做空 | **強烈看多**（散戶做空 = 反向確認） |
| ❌ 減倉 | ✅ 做多 | **看空警訊**（散戶追高 = 可能頂部） |
| ⚠️ 觀望 | ✅ 單邊做多 | **中性偏空**（散戶過度樂觀需謹慎） |

**資產類別映射**：

| 報告中的分類 | 對應本模組的資產 |

**評級轉換為分數**：

| 評級 | 分數 |
|---|---|
| OW (Overweight) | +1 |
| N (Neutral) | 0 |
| UW (Underweight) | -1 |

**計算「機構平均分數」**：

```
機構平均分數 = (PIMCO 分數 + JP Morgan 分數 + BlackRock 分數) / 3
```

### 步驟 5：計算「資金溫度分數」（-2 ～ +2）

**目標**：綜合「ETF 資金流 + 價格動能 + COT 大戶持倉 + 機構評級」，給每個資產一個統一評分

#### 5.1 評分公式（四維加權）

```
資金溫度分數 =
(ETF 資金流分數 × 0.35) +
(價格動能分數 × 0.25) +
(機構評級分數 × 0.25) +
(COT 大戶信號 × 0.15)
```

**權重設計理由**：

| 指標 | 權重 | 理由 |
|---|---|---|
| ETF 資金流 | 35% | 最直接的現貨資金流，即時性最高 |
| 價格動能 | 25% | 驗證資金流是否有效推動價格 |
| 機構評級 | 25% | 反映長期戰略配置方向 |
| COT 大戶信號 | 15% | 期貨市場領先指標，驗證專業投資者部位 |

---

#### 5.2 最終分數處理

將加權計算結果四捨五入到以下離散值：

```
-2, -1.5, -1, -0.5, 0, +0.5, +1, +1.5, +2
```

**計算範例（黃金 GLD）**：

```
ETF 資金流：淨流入 $900M → +1 × 0.35 = 0.35
價格動能：近期上漲 → +1 × 0.25 = 0.25
機構評級：3 家均 OW → +2 × 0.25 = 0.50
COT 信號：強烈做多 → +0.5 × 0.15 = 0.075

資金溫度分數 = 0.35 + 0.25 + 0.50 + 0.075 = 1.175
→ 四捨五入為 +1.5
```

## 四、文字總結與投資啟示

### 4.1 資金流向總結（三重驗證結論）

本週跨資產資金流呈現明顯「避險輪動 + 價值回歸」特徵：

1. **資金大幅撤出的資產**：
   - **美股大型科技（QQQ）**：ETF 淨流出 $800M + COT 期貨淨多單減少 + 估值過高，**三重確認**資金撤出。

   - **投資級債券（AGG）+ 美國公債 10 年（IEF）**：ETF 資金流入 + COT 期貨做多 + 機構一致 OW，**三重確認**債券牛市啟動。
   - **黃金（GLD）+ 白銀（SLV）**：ETF 淨流入 + COT 大戶連續加碼 + 價格創新高 + 機構一致 OW，**四重確認**貴金屬進入長期上升趨勢。

### 4.2 機構共識解讀

**三大機構（PIMCO、JP Morgan、BlackRock）一致 Overweight 的資產：**
- 投資級債券
- 美國公債 10 年
- 黃金

**共識解讀：**
- 機構普遍認為 **Fed 降息週期已啟動或即將啟動**，債券進入戰略配置視窗。
- **通膨預期回升 + 地緣風險 + 去美元化**，推動黃金成為「新避險之王」。

### 4.3 對台股／亞股的啟示（5 點）

1. **電子權值股短線承壓**：美股科技股資金外流 + COT 期貨大戶減倉，將連帶影響台股高價電子。

2. **原物料供應鏈受惠**：黃金、白銀資金大幅湧入，台灣稀土概念股、貴金屬回收可能受惠。

3. **高股息防禦股獲青睞**：美股價值股資金淨流入，台股高股息 ETF 可能跟隨輪動。

4. **留意美債殖利率走勢**：若美債殖利率持續下行，將降低台股的「相對吸引力折價」，有利外資回流。

5. **美元貶值利多出口**：美元指數 COT 大戶做空，若美元持續貶值，將有利台灣出口企業。

### 4.4 風險提示

1. **COT 數據延遲性**：COT 報告每週五發布（統計截至週二），存在 3 天延遲。
2. **機構評級更新頻率**：部分機構報告為季度或月度更新，若本週無新報告，評級可能不代表最新觀點。
3. **本報告不構成投資建議**：僅提供資金流分析框架，實際投資需結合個人風險承受度。

## 五、在第一階段的定位

**本模組在投資決策流程中的位置**：

```
第一階段：總體經濟分析
├── 1A. 經濟指標解讀（GDP、CPI、利率、就業）
├── 1B. 央行政策評估（Fed 決策、流動性）
└── 1C. 跨資產資金流追蹤 ← 本模組
     ↓
     判斷：現在該配置股票 or 債券 or 現金？
     ↓
（若資金流顯示「股票大幅淨流出」，則不進入第二階段選產業）
     ↓
第二階段：產業研究與輪動
第三階段：個股基本面分析
...
```

**為什麼放在第一階段？**

1. **邏輯順序正確**：先判斷「該不該投資股票」，再選產業、選個股
2. **功能定位清晰**：跨資產配置屬於宏觀決策，不是個股情緒
3. **與現有模組互補**：與經濟指標、央行政策形成完整的總體分析三角
4. **提升決策效率**：避免在錯誤的時機點花大量時間分析個股

## 六、與其他模組的配合

### 與第一階段其他模組的協同

**整合分析流程**：

```
Step 1: 經濟指標解讀（economic-indicators.md）
↓ 判斷經濟處於哪個週期階段

Step 2: 央行政策評估（fed-policy-framework.md）
↓ 判斷政策立場（鴿派/鷹派）

Step 3: 跨資產資金流追蹤（本模組）
↓ 判斷資金實際流向哪些資產類別

綜合結論
↓ 產出資產配置建議（股 X% / 債 Y% / 現金 Z%）
```

**一致性驗證範例**：

```
經濟指標：經濟放緩 + 通膨回落 → 預期 Fed 降息
央行政策：Fed 官員發言轉鴿派 → 市場預期降息
資金流向：債券 ETF 大幅淨流入 + COT 大戶做多公債 → 確認

→ 三者一致：高信度看好債券，建議增持債券至 40%
```

**矛盾訊號處理**：

```
經濟指標：經濟放緩 → 預期降息利多債券
央行政策：Fed 堅持鷹派 → 利率維持高檔
資金流向：債券 ETF 淨流出 → 市場不看好債券

→ 訊號矛盾：降低部位，保留現金觀望
```

## 七、注意事項與最佳實踐

### 7.1 數據時效性

- **ETF 資金流**：通常延遲 1-2 天
- **COT 報告**：每週五發布（統計截至週二），存在 3 天延遲
- **機構報告**：通常每季或每月更新

### 7.2 數據可得性

- 部分資產（如 VTV、AGG、HYG）無對應的 COT 期貨合約
- 若查不到某資產的 COT 數據，該項分數記為 0，權重分配至其他三項

### 7.3 三重驗證的侷限性

即使三者一致，市場仍可能因突發事件（如地緣衝突、央行政策突變）快速反轉。建議搭配技術分析、基本面研究與風險管理工具使用。

### 7.4 本模組的定位

**本模組僅提供資金流分析框架**，不構成個股或槓桿產品的具體投資建議。實際投資需結合個人風險承受度、財務狀況與投資目標。
