# Whale Watching 巨鯨追蹤指南

## 概述

Whale Watching(巨鯨追蹤)是指追蹤市場上擁有龐大資金的「大戶」交易行為,包括機構投資者、對沖基金、超級富豪等。巨鯨的一舉一動往往領先散戶,跟對巨鯨,就像衝浪跟對大浪,能乘風破浪獲利豐厚。

---

## 誰是市場巨鯨?

### 巨鯨分類

| 類型 | 定義 | 資金規模 | 影響力 | 可追蹤性 |
|------|------|---------|--------|---------|
| **機構投資者** | 共同基金、養老基金 | 百億-兆級 | ⭐⭐⭐⭐⭐ | 高(13F) |
| **對沖基金** | 激進投資、高槓桿 | 十億-百億 | ⭐⭐⭐⭐⭐ | 高(13F) |
| **超級富豪** | 個人大戶(巴菲特等) | 十億-千億 | ⭐⭐⭐⭐ | 中(需推測) |
| **企業自營** | 公司回購、庫藏股 | 數億-百億 | ⭐⭐⭐⭐ | 高(公告) |
| **國家主權基金** | 挪威石油基金、新加坡 GIC | 兆級 | ⭐⭐⭐⭐ | 低(不透明) |
| **內部人** | CEO、董事會 | 數億-數十億 | ⭐⭐⭐⭐ | 高(Form 4) |

### 追蹤優先順序

```

1. 明星對沖基金經理(巴菲特、Burry、Ackman)
2. 內部人集體交易(多位高管同時買賣)
3. 企業回購計畫
4. 大型機構季度調倉(13F)
5. Dark Pool 異常交易
```

## 巨鯨追蹤數據來源

### 1. SEC Form 13F(美股機構持倉)

**定義:**
管理資產 > $100M 的機構每季需申報持股明細。

**公布時間:**
- 季度結束後 45 天內
- 2月中、5月中、8月中、11月中

**可查詢內容:**
- 持股標的與數量
- 持股變化(增持/減持)
- 新增/清倉標的

**查詢網址:**
- SEC EDGAR: https://www.sec.gov/cgi-bin/browse-edgar
- WhaleWisdom: https://whalewisdom.com/
- DataRoma: https://www.dataroma.com/m/home.php

**限制:**
- ⚠️ 45 天時間差(數據滯後)
- ⚠️ 僅多頭部位(不含放空)
- ⚠️ 不含非美股資產

### 2. SEC Form 4(內部人交易)

**定義:**
公司內部人(高管、董事、持股 > 10% 股東)需在交易後 2 個工作日內申報。

**即時性:**
- ✅ 高(2 日內申報)
- ✅ 比 13F 更即時

**查詢網址:**
- OpenInsider: http://openinsider.com/
- SEC EDGAR: https://www.sec.gov/ (搜尋 Form 4)
- Finviz Insider: https://finviz.com/insidertrading.ashx

### 3. Dark Pool Transactions(暗池交易)

**定義:**
機構私下進行的大額交易,不在公開市場顯示,避免影響股價。

**特徵:**
- 單筆 > 10,000 股
- 不公開顯示在 Order Book
- 交易後才公布

**查詢方式:**
- Quiver Quantitative: https://www.quiverquant.com/
- Finra OTC Data: https://otctransparency.finra.org/
- Finviz Elite: https://finviz.com/elite.ashx

### 4. 公司回購計畫

**定義:**
公司在公開市場買回自家股票。

**為什麼重要:**
- 公司認為股價被低估
- 減少流通股數,提高 EPS
- 通常是看多信號

**查詢方式:**
- 公司公告(8-K)
- Birinyi Associates(專業追蹤回購)
- TrimTabs(回購數據庫)

## 巨鯨行為解讀

### 1. 13F 持倉變化分析

**新增持股(New Position):**

| 情境 | 金額 | 含義 | 操作建議 |
|------|------|------|---------|
| **巴菲特新增持股** | > $1B | 極度看好 | 🟢 深入研究,可能長期持有 |
| **多家頂級基金新增** | > $500M | 機構共識看好 | 🟢 跟進 |
| **單一基金新增** | < $100M | 試探性建倉 | 🟡 觀察後續 |

**大幅加倉(Increased Position):**

| 增幅 | 含義 | 操作建議 |
|------|------|---------|
| **> 100%** | 加倍持股,極度看好 | 🟢 強烈看多信號 |
| **50-100%** | 明顯加碼 | 🟢 看多 |
| **25-50%** | 正常加碼 | 🟢 正面 |
| **< 25%** | 小幅調整 | 🟡 中性 |

**大幅減倉(Decreased Position):**

| 減幅 | 含義 | 操作建議 |
|------|------|---------|
| **> 75%** | 幾乎清倉,強烈看淡 | 🔴 警訊 |
| **50-75%** | 大幅減碼 | 🔴 看空 |
| **25-50%** | 正常減碼 | 🟡 謹慎 |
| **< 25%** | 小幅調整 | 🟡 中性 |

**完全清倉(Sold Out):**

| 情境 | 含義 | 操作建議 |
|------|------|---------|
| **巴菲特清倉** | 基本面可能轉差 | 🔴 重新評估該標的 |
| **多家基金清倉** | 機構共識看淡 | 🔴 考慮退出 |

### 2. 內部人交易解讀

**內部人買進(Insider Buying):**

**最強信號組合:**
```

✅ CEO + CFO + 多位董事同時買進
✅ 買進金額 > \$100K(有意義金額)
✅ 非例行性操作(非 10b5-1 計畫)
✅ 股價處於低點或整理
→ 🟢 內部人極度看好

```

**單一高管買進:**
```

金額 < \$50K
→ 🟡 可能只是象徵性買進,參考價值低

```

**內部人賣出(Insider Selling):**

**警訊組合:**
```

🔴 多位高管同時大額賣出(> \$1M)
🔴 非計畫性賣出(非 10b5-1)
🔴 賣出時間接近財報或重大事件
→ 🔴 可能知道利空

```

**正常賣出:**
```

🟡 例行性 10b5-1 計畫賣出
🟡 行使選擇權後部分賣出(套現)
🟡 分散投資(賣出持股 < 25%)
→ 🟡 不一定看空,可能只是資產配置

```

**統計規律:**
- 內部人買進後 6 個月平均報酬: +5-8%
- 內部人大額賣出後 6 個月平均報酬: -2-5%

### 3. Dark Pool 交易解讀

**大額買進(Bullish):**

```

Dark Pool 單日買進 > 500萬股
+
佔當日總成交量 > 20%
→ 機構大舉買進,避免推高股價
→ 🟢 強烈看多信號

```

**大額賣出(Bearish):**

```

Dark Pool 單日賣出 > 500萬股
+
佔當日總成交量 > 20%
→ 機構大舉出貨
→ 🔴 強烈看空信號

```

**配合 Options Flow:**

**最強組合:**
```

Dark Pool 大量買進
+
大額 Call Golden Sweep
→ 機構全方位做多
→ 🟢🟢🟢 最強看多信號

```

### 4. 企業回購解讀

**回購計畫宣布:**

| 回購金額 | 相對市值 | 含義 | 操作建議 |
|---------|---------|------|---------|
| **> 10% 市值** | 極大 | 公司極度看好自家股票 | 🟢 強烈看多 |
| **5-10% 市值** | 大 | 明顯看好 | 🟢 看多 |
| **2-5% 市值** | 中 | 正常回購 | 🟡 正面 |
| **< 2% 市值** | 小 | 象徵性 | 🟡 影響有限 |

**實際執行率:**

**關鍵:**
宣布回購 ≠ 實際執行,需追蹤執行進度。

| 執行率 | 含義 | 操作建議 |
|-------|------|---------|
| **> 80%** | 積極執行,真心看好 | 🟢 看多 |
| **50-80%** | 正常執行 | 🟡 正面 |
| **< 50%** | 執行不積極,可能只是做秀 | 🔴 警惕 |

**案例:**

**Apple (AAPL) 回購:**
```

2023年宣布 \$900億回購計畫
相對市值: 3%
執行率: 90%+
→ 積極回購,支撐股價
→ 成為股價上漲主要動力之一

```

## 巨鯨追蹤實戰策略

### 策略 1: 跟隨巴菲特(Buffett Copycat)

**原理:**
巴菲特選股能力極強,跟隨其持倉長期報酬優異。

**執行步驟:**

**Step 1: 等待 13F 公布**
- 時間:每季度結束後 45 天
- 網址: https://www.dataroma.com/m/holdings.php?m=BRK

**Step 2: 識別重大變化**
```

✅ 新增持股(New Position)
✅ 大幅加倉 > 50%
✅ 成為前 10 大持股
→ 巴菲特高度看好

```

**Step 3: 深入研究**
- 用第三階段分析基本面
- 用第四階段評估估值
- 確認邏輯後跟進

**Step 4: 長期持有**
- 巴菲特持股週期長(數年)
- 不短線進出
- 定期檢視基本面

**歷史績效:**
- 跟隨巴菲特持股組合年化報酬: +12-15%
- 略低於巴菲特(有時間差),但優於大盤

**成功案例:**

**Bank of America (BAC):**
```

2017-2019: 巴菲特持續加碼
→ 成為 BRK 第二大持股
跟隨者買進 @ \$25-30
2021年高點: \$48
報酬: +60-92%

```

**比亞迪(BYD,港股):**
```

2008: 巴菲特買進
持有 14 年
2022開始減持
期間報酬: > 3,000%

```

### 策略 2: 內部人集體買進

**原理:**
多位內部人同時買進 = 極度看好公司前景。

**篩選條件:**
```

1. 3 位以上高管同時買進
2. 單筆金額 > \$100K
3. 非例行性 10b5-1 計畫
4. 過去 30 天內發生
5. 股價處於低點(跌幅 > 20%)
6. 基本面未嚴重惡化
→ 內部人集體看好,逆向機會
```

**執行:**
- 配置 30-40% 倉位
- 持有期: 6-12 個月
- 停利: +30% 或內部人開始賣出
- 停損: -10%

**歷史勝率:** 約 70-75%

**案例:**

**某生技公司 - 2022年:**
```

背景:

- 臨床試驗失敗,股價暴跌 -60%
- 從 \$50 跌至 \$20

內部人動作:

- CEO 買進 \$2M
- CFO 買進 \$500K
- 3 位董事各買 \$300K
- 總計 \$3.4M
→ 內部人強烈信心

結果:

- 6 個月後推出新藥
- 股價反彈至 \$45(+125%)

```

### 策略 3: Dark Pool + Options Flow 組合

**原理:**
機構同時在現貨與選擇權市場布局,最強信號。

**買進條件:**
```

1. Dark Pool 單日買進 > 500萬股
2. 佔總成交量 > 20%
3. 同日出現大額 Call Golden Sweep(> \$1M)
4. 股價技術面突破關鍵阻力
5. 基本面良好或有催化劑
→ 機構重注做多
```

**執行:**
- 配置 30-50% 倉位
- 持有期: 2-4 週(短中期)
- 停利: +20% 或 Dark Pool 轉為賣出
- 停損: -8%

**歷史勝率:** 約 65-70%

**案例:**

**NVIDIA (NVDA) - 2023年5月:**
```

5/22 (AI 熱潮啟動):

- Dark Pool 買進 800萬股(\$3.2B)
- Call Golden Sweep:
    - 450C \$25M
    - 500C \$18M
    - 520C \$12M
- 合計 \$55M Call + \$3.2B 股票
→ 機構史詩級做多

操作:

- 次日買進 @ \$380
- 3 週後 \$480 (+26%)
- 3 個月後 \$500 (+32%)

```

### 策略 4: 企業回購加速期

**原理:**
公司積極回購時,為股價提供支撐與買盤。

**判斷條件:**
```

1. 宣布 > 5% 市值回購計畫
2. 實際執行率 > 80%
3. 股價處於低點或整理
4. 基本面穩健
5. 現金流充足(能持續回購)
→ 回購支撐股價
```

**執行:**
- 配置 40-50% 倉位
- 持有期: 6-12 個月
- 停利: +25% 或回購計畫結束
- 停損: -10%

**歷史勝率:** 約 70%

**案例:**

**Meta (META) - 2023年:**
```

背景:

- 2022年元宇宙燒錢,股價崩跌
- 從 \$380 跌至 \$90(-76%)

回購計畫:

- 2023年宣布 \$400億回購
- 積極執行,每季 \$50-80億
- 執行率 > 90%

結果:

- 回購 + 降本增效
- 股價 \$90 → \$350(+289%)

```

## 巨鯨追蹤進階技巧

### 1. 識別假巨鯨(Fake Whales)

**問題:**
不是所有大額交易都值得跟隨。

**假巨鯨特徵:**

**ETF 被動調倉:**
```

🟡 指數 ETF(SPY、QQQ)定期 Rebalance
→ 非主動看多/看空,只是被動調整
→ 參考價值低

```

**機構避險交易:**
```

🟡 機構買進股票 + 賣出 Call
→ 收取權利金策略,非看多
→ 不是純粹做多信號

```

**強制平倉:**
```

🟡 機構因虧損被迫平倉
→ 非主動賣出,是被動行為
→ 不代表看空

```

**辨別方法:**
- 確認交易目的(主動 vs 被動)
- 觀察是否有配套操作(買股 + 買 Call = 真看多)
- 追蹤後續動作(是否持續加碼)

### 2. 巨鯨行為的時間差

**13F 的 45 天滯後:**

**問題:**
13F 公布時,巨鯨可能已改變策略。

**案例:**
```

某基金 Q1 新增持股 XYZ
→ 13F 在 5/15 公布
→ 但基金可能在 4/30 已賣出
→ 散戶 5/15 跟進,買在高點

```

**解決方案:**
- 配合近期股價與新聞判斷
- 若 13F 公布後股價已大漲 > 20%, 謹慎追高
- 優先追蹤「連續數季」加碼的標的

### 3. 反向追蹤:巨鯨賣什麼

**原理:**
巨鯨賣出的標的,可能基本面惡化。

**警訊信號:**
```

🔴 多家頂級基金同時清倉
🔴 巴菲特清倉持股多年的標的
🔴 內部人集體大額賣出
→ 可能知道散戶不知道的利空

```

**操作:**
- 重新評估該標的
- 若持有,考慮減碼或退出
- 避免「逢低加碼」陷阱

**案例:**

**富國銀行(WFC):**
```

2020-2022: 巴菲特持續減持 WFC
→ 從第一大持股降至清倉
→ 原因:監管問題、醜聞不斷

散戶若注意到此訊號:
→ 及時退出,避免後續下跌

```

## 台股巨鯨追蹤

### 1. 外資買賣超(最重要)

**追蹤方式:**
- 證交所三大法人買賣超
- 個股外資持股比例變化

**信號強度:**

| 外資動作 | 持續時間 | 信號強度 | 操作建議 |
|---------|---------|---------|---------|
| **連買 10 天+** | 長期 | ⭐⭐⭐⭐⭐ | 強烈看多 |
| **單日買超 > 50億** | 單日 | ⭐⭐⭐⭐ | 短期看多 |
| **持股比例上升 > 5%** | 季度 | ⭐⭐⭐⭐ | 長期看多 |
| **連賣 10 天+** | 長期 | ⭐⭐⭐⭐⭐ | 強烈看空 |

**詳細解讀請參考:** `institutional-flow-taiwan.md`

### 2. 董監持股變化

**申報時間:**
- 每月 10 日前申報上月持股變化

**重點觀察:**

**大幅增持:**
```

董監事持股增加 > 10%
+
多位董事同時增持
→ 內部人看好
→ 🟢 正面信號

```

**大幅減持:**
```

董監事持股減少 > 30%
+
持股比例降至低點
→ 內部人套現
→ 🔴 警訊

```

### 3. 庫藏股買回

**定義:**
公司在公開市場買回自家股票。

**實施原因:**
- 轉讓給員工
- 維護股價
- 註銷(提升 EPS)

**判斷標準:**

| 買回比例 | 執行率 | 含義 | 操作建議 |
|---------|-------|------|---------|
| **> 5% 股本** | > 80% | 積極護盤,看好 | 🟢 看多 |
| **2-5% 股本** | > 60% | 正常回購 | 🟡 正面 |
| **< 2% 股本** | < 50% | 象徵性,效果有限 | 🟡 影響小 |

## 結論

巨鯨追蹤是「站在巨人肩膀上」的投資策略,跟對了事半功倍。

**核心原則:**
1. ✅ **巴菲特新增/大幅加碼 = 值得深入研究**
2. ✅ **內部人集體買進 = 強烈信心**
3. ✅ **Dark Pool 大買 + Call Sweep = 最強組合**
4. ✅ **企業積極回購 = 股價支撐**
5. ✅ **多家巨鯨同時清倉 = 警訊**

**優先順序:**
```

明星投資人(巴菲特) > 內部人集體交易 > Dark Pool > 一般機構
連續動作 > 單次動作
大額交易(> \$500M) > 小額調整

```

**注意事項:**
- 13F 有 45 天時間差,需謹慎
- 巨鯨也會犯錯,需配合基本面判斷
- 不盲目跟單,理解邏輯後再行動
- 設定停損,巨鯨不保證獲利
