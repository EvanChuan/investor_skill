# Social Sentiment 社群情緒分析

## 概述

Social Sentiment(社群情緒)透過分析 Reddit、Twitter、StockTwits 等社群媒體的討論內容,量化散戶情緒與熱度。在 GameStop、AMC 等 Meme Stock 事件後,社群情緒已成為不可忽視的市場力量。本文檔詳細說明如何追蹤與應用社群情緒數據。

---

## 社群情緒基礎知識

### 為什麼社群情緒重要?

**散戶力量崛起:**
- 2021 年 GME 事件證明散戶集結可撼動市場
- 社群媒體成為散戶主要資訊來源
- Meme Stock 現象:情緒 > 基本面

**領先指標:**
- 社群討論度往往領先股價 1-3 天
- 極端情緒(過度樂觀/恐慌)預示反轉
- 熱度爆發 = 短線交易機會

**風險指標:**
- 過度討論 = 散戶 FOMO,可能見頂
- 無人問津 = 價值被忽視,逆向機會

**如何追蹤:**
- 觀察「熱門帖子」討論標的

```

🚀🚀🚀 = 極度看多,準備飆漲
💎🙌 = Diamond Hands,死抱不賣
```

## 社群情緒核心指標

### 1. Mentions(提及次數)

**定義:**
某股票在特定時間內被提及的總次數。

**解讀標準(以 WSB 為例):**

| Mentions(24h) | 熱度 | 市場狀態 | 操作建議 |
|--------------|------|---------|---------|
| **> 10,000** | 極度瘋狂 | 散戶 FOMO,過熱 | 🔴 考慮獲利或避開 |
| **5,000-10,000** | 高度關注 | 熱門標的 | 🟡 警戒,可能見頂 |
| **1,000-5,000** | 中度關注 | 正常熱度 | 🟢 值得研究 |
| **500-1,000** | 初期關注 | 開始萌芽 | 🟢 早期機會 |
| **< 500** | 低度關注 | 無人問津 | 🟢 可能被低估(逆向) |

**變化趨勢:**

| 變化 | 含義 | 操作建議 |
|------|------|---------|
| **暴增 > 500%** | 爆紅,FOMO | 🔴 警戒,可能追高 |
| **持續增加** | 熱度升溫 | 🟡 關注,準備進場 |
| **持平** | 正常關注 | 🟢 穩定 |
| **快速下降** | 熱度退潮 | 🔴 可能見頂,考慮退出 |

### 2. Sentiment Score(情緒分數)

**定義:**
透過自然語言處理(NLP)分析貼文,計算看多/看空情緒。

**計算方式:**
```

Sentiment Score = (看多貼文數 - 看空貼文數) / 總貼文數 × 100

範圍: -100(極度看空) ~ +100(極度看多)

```

**解讀標準:**

| Sentiment Score | 情緒 | 市場特徵 | 操作建議 |
|----------------|------|---------|---------|
| **> +80** | 極度樂觀 | 散戶狂熱,危險 | 🔴 考慮賣出 |
| **+60~+80** | 高度樂觀 | 看多為主 | 🟡 謹慎持有 |
| **+30~+60** | 偏向樂觀 | 正常看多 | 🟢 持有 |
| **-30~+30** | 中性 | 多空分歧 | 🟡 觀望 |
| **-60~-30** | 偏向悲觀 | 正常看空 | 🟡 謹慎 |
| **-80~-60** | 高度悲觀 | 看空為主 | 🟢 逆向機會 |
| **< -80** | 極度悲觀 | 散戶恐慌 | 🟢 強烈逆向機會 |

### 3. Message Volume Growth(討論量增長率)

**定義:**
```

增長率 = (今日討論量 - 昨日討論量) / 昨日討論量 × 100%

```

**解讀標準:**

| 增長率 | 熱度變化 | 含義 | 操作建議 |
|-------|---------|------|---------|
| **> +200%** | 爆發式增長 | 重大事件/爆紅 | 🟡 警戒,可能追高 |
| **+50~200%** | 快速升溫 | 關注度提升 | 🟢 早期機會 |
| **+10~50%** | 穩定增長 | 正常升溫 | 🟢 正面 |
| **-10~+10%** | 持平 | 穩定關注 | 🟡 中性 |
| **-50~-10%** | 降溫 | 熱度退潮 | 🔴 警訊 |
| **< -50%** | 快速降溫 | 無人關注 | 🔴 可能見頂 |

### 4. Bullish/Bearish Ratio(看多看空比)

**定義(StockTwits):**
```

Bullish Ratio = 標記看多的用戶 / 總標記用戶 × 100%

```

**解讀標準:**

| Bullish % | 情緒 | 特徵 | 操作建議 |
|-----------|------|------|---------|
| **> 90%** | 一面倒看多 | 極度樂觀,危險 | 🔴 逆向賣出 |
| **70-90%** | 高度看多 | 散戶樂觀 | 🟡 謹慎 |
| **50-70%** | 偏多 | 正常看多 | 🟢 持有 |
| **30-50%** | 偏空 | 正常看空 | 🟡 謹慎 |
| **10-30%** | 高度看空 | 散戶悲觀 | 🟢 逆向機會 |
| **< 10%** | 一面倒看空 | 極度悲觀 | 🟢 強烈逆向機會 |

## 社群情緒交易策略

### 策略 1: 逆向操作(Contrarian)

**原理:**
當散戶情緒極端時,往往是反轉信號。

**極度樂觀時賣出:**
```

條件:

1. WSB Mentions > 10,000(24h)
2. Sentiment Score > +80
3. Bullish % > 90%
4. 討論量暴增 > 500%
5. 到處都是 🚀🚀🚀
→ 散戶 FOMO,考慮賣出
```

**執行:**
- 獲利了結 50-70%
- 或買入 Put 保護
- 等待回調再進場

**極度悲觀時買進:**
```

條件:

1. Sentiment Score < -60
2. Bullish % < 20%
3. 討論量極低(無人關注)
4. 股價下跌 > 20%
5. 基本面未惡化
→ 散戶恐慌,逆向買進
```

**執行:**
- 分批買進(3批)
- 配置 30-50% 倉位
- 停損: -10%

**歷史勝率:** 約 70%(需配合基本面)

### 策略 2: 趨勢跟隨(Momentum)

**原理:**
社群熱度初期上升時,股價往往跟隨。

**買進條件:**
```

1. Mentions 從 < 500 增至 1,000-2,000
2. Sentiment Score > +50
3. 討論量連續 3 天增長 > 50%
4. 尚未爆紅(< 5,000 Mentions)
5. 有明確催化劑(財報、新產品)
→ 熱度初期,趨勢跟隨
```

**執行:**
- 買進 30-40% 倉位
- 停利: Mentions > 10,000 或漲幅 > 30%
- 停損: -8%

**賣出條件:**
```

1. Mentions 達到 > 10,000(過熱)
2. Sentiment Score > +85(極度樂觀)
3. 討論量開始下降
→ 熱度見頂,獲利了結
```

**歷史勝率:** 約 60-65%

### 策略 3: 早期發現(Early Detection)

**原理:**
在標的爆紅前捕捉訊號。

**篩選條件:**
```

1. Mentions 從 < 100 突增至 500-1,000
2. 增長率 > 500%
3. Sentiment Score > +60
4. 有明確催化劑
5. 市值 < \$5B(小型股易炒作)
6. Short Interest > 15%(軋空潛力)
→ 潛在爆紅標的
```

**執行:**
- 小倉位試單 10-20%
- 密切監控熱度變化
- 停利: 爆紅(Mentions > 5,000)或漲幅 > 50%
- 停損: -12%(波動大)

**風險:**
- ⚠️ 可能不爆紅,空等
- ⚠️ 可能是 Pump & Dump 騙局
- ⚠️ 波動極大

**歷史勝率:** 約 50-55%(高風險高報酬)

## Meme Stock 識別與操作

### Meme Stock 特徵

**定義:**
由社群媒體推動,脫離基本面的投機標的。

**典型特徵:**
```

✅ WSB Mentions > 5,000
✅ Sentiment 極度樂觀(> +75)
✅ 社群到處是 🚀🚀🚀
✅ Short Interest 高(> 15%)
✅ 股價短期暴漲(> 100%)
✅ 基本面普通或虧損
✅ 新聞媒體跟進報導

```

**經典 Meme Stocks:**
- GameStop (GME)
- AMC Entertainment (AMC)
- BlackBerry (BB)
- Bed Bath & Beyond (BBBY)
- Sundial Growers (SNDL)

### Meme Stock 生命週期

**Phase 1: 萌芽期(1-3天)**
```

- Mentions 500-2,000
- 小部分用戶開始討論
- 股價小幅上漲(+5~15%)
→ 🟢 最佳進場點

```

**Phase 2: 升溫期(3-7天)**
```

- Mentions 2,000-5,000
- 討論度快速攀升
- 股價明顯上漲(+20~50%)
- 新聞開始報導
→ 🟡 可持有,設好停利

```

**Phase 3: 狂熱期(7-14天)**
```

- Mentions > 10,000
- 散戶 FOMO,瘋狂追高
- 股價暴漲(+100~500%)
- 到處都是 🚀🚀🚀
→ 🔴 見頂信號,獲利了結

```

**Phase 4: 崩潰期(突然)**
```

- Mentions 快速下降
- Sentiment 轉負
- 股價暴跌(-50~80%)
- 散戶套牢哀嚎
→ 🔴 已來不及,避免接刀

```

**操作建議:**
- ✅ Phase 1 進場(早期)
- ✅ Phase 2 持有(設停利)
- 🔴 Phase 3 獲利了結(不貪心)
- 🔴 Phase 4 不接刀(等熱度完全消退)

## 社群情緒陷阱與風險

### 陷阱 1: Pump & Dump 騙局

**定義:**
有組織地拉高股價後出貨,散戶套牢。

**識別特徵:**
```

🔴 低市值標的(< \$500M)
🔴 突然爆紅(Mentions 暴增 > 1,000%)
🔴 來源可疑(新 Discord 群、匿名 Telegram)
🔴 情緒極度樂觀(+95)
🔴 無明確催化劑
🔴 基本面極差(虧損、負債)
→ 可能是 Pump \& Dump,避開!

```

**案例:**
```

某 Penny Stock:

- 市值 \$200M
- 突然 WSB 爆紅(Mentions 5,000)
- 股價 3 天暴漲 +400%
- 隨後 5 天暴跌 -80%
→ 典型 Pump \& Dump

```

### 陷阱 2: 回音室效應(Echo Chamber)

**定義:**
社群內只有單一聲音,忽視反面意見。

**特徵:**
```

- 任何看空意見被攻擊
- "Diamond Hands"被過度鼓吹
- 理性分析被視為 FUD(Fear, Uncertainty, Doubt)
→ 危險信號,可能集體套牢

```

**案例:**
GME 狂熱期,任何建議獲利的人被罵"紙手"(Paper Hands),導致大量散戶套在高點。

### 陷阱 3: 滯後性

**問題:**
社群情緒反映的是「已發生」的事,不是「將發生」。

**範例:**
```

股價已漲 +100%
→ 社群才開始瘋狂討論
→ 散戶此時追高
→ 套牢

```

**解決方案:**
- 關注「早期討論」(Mentions < 2,000)
- 不追已爆紅標的(Mentions > 10,000)
