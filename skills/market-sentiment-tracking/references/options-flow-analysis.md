# Options Flow 選擇權流向分析

## 概述

Options Flow(選擇權流向)追蹤大額選擇權交易,揭示機構與聰明錢的真實意圖。與散戶不同,機構不會公開宣布他們的看法,但會透過選擇權市場悄悄布局。學會解讀 Options Flow,就像竊聽華爾街的內部通話。

---

## Options Flow 基礎知識

### 什麼是 Options Flow?

**定義:**
即時追蹤選擇權市場的大額交易,包括:
- 交易方向(買進/賣出)
- 選擇權類型(Call/Put)
- 執行價(Strike Price)
- 到期日(Expiration)
- 交易金額(Premium)

**為什麼重要?**
- 📊 機構不會大量買股票(會推升股價)
- 📊 機構會用選擇權建立部位(隱密、槓桿高)
- 📊 大額選擇權單通常領先股價 1-7 天

### Options Flow vs Put/Call Ratio

| 指標 | Options Flow | Put/Call Ratio |
|------|-------------|----------------|
| **粒度** | 單筆交易級別 | 整體市場統計 |
| **即時性** | 即時 | 每日更新 |
| **用途** | 捕捉大戶動向 | 判斷整體情緒 |
| **優勢** | 精準,可追蹤個股 | 簡單,容易判斷 |
| **難度** | 高(需解讀) | 低 |

## Options Flow 核心概念

### 1. 交易方向判斷

**買方 vs 賣方:**

| 交易方式 | 代碼 | 含義 | 市場意圖 |
|---------|------|------|---------|
| **Buy to Open** | BTO | 買進開倉 | 建立新多頭/空頭部位 |
| **Sell to Open** | STO | 賣出開倉 | 賣方收取權利金 |
| **Buy to Close** | BTC | 買進平倉 | 回補先前賣出部位 |
| **Sell to Close** | STC | 賣出平倉 | 平倉先前買入部位 |

**如何判斷買賣方?**

**看 Bid/Ask 位置:**
```

交易價接近 Ask = 買方主動(Aggressive Buy)
→ 買方急著進場,看多/看空強烈

交易價接近 Bid = 賣方主動(Aggressive Sell)
→ 賣方急著出場或收權利金

```

**範例:**
```

AAPL 180 Call
Bid: \$2.50, Ask: \$2.55
成交價: \$2.54
→ 接近 Ask,買方主動 = 看多訊號

```

### 2. Call vs Put 解讀

**基本邏輯:**

| 交易 | 方向 | 看法 | 強度 |
|------|------|------|------|
| **Buy Call** | 做多 | 看漲 | ⬆️⬆️⬆️ |
| **Sell Put** | 做多 | 看漲(收權利金) | ⬆️⬆️ |
| **Buy Put** | 做空 | 看跌 | ⬇️⬇️⬇️ |
| **Sell Call** | 做空 | 看跌(收權利金) | ⬇️⬇️ |

**進階組合:**

**牛市價差(Bull Spread):**
```

Buy Call(低執行價) + Sell Call(高執行價)
→ 看漲但有限度,降低成本

```

**熊市價差(Bear Spread):**
```

Buy Put(高執行價) + Sell Put(低執行價)
→ 看跌但有限度,降低成本

```

### 3. 到期日(Expiration)判斷

**時間框架:**

| 到期日 | 持有期 | 特性 | 交易者類型 |
|-------|--------|------|-----------|
| **0-7 DTE** | 本週 | 極短線賭博 | 散戶投機客 |
| **7-30 DTE** | 本月 | 短線波段 | 活躍交易者 |
| **30-90 DTE** | 1-3個月 | 中線佈局 | 機構避險/投機 |
| **> 90 DTE** | 長期 | 長線佈局/避險 | 機構長期部位 |

**解讀重點:**

```

短期選擇權(< 30 DTE):
→ 預期短期內有重大事件(財報、FDA、併購)
→ 時間敏感,需密切關注

長期選擇權(> 90 DTE):
→ 機構長期看法
→ 參考價值更高(機構不做短線賭博)

```

### 4. 執行價(Strike Price)判斷

**價內/價外:**

| 執行價位置 | 代碼 | 特性 | 勝率 | 報酬 |
|-----------|------|------|------|------|
| **深度價內** | Deep ITM | Delta 接近 1,像股票 | 高 | 低 |
| **價內** | ITM | 內含價值 > 0 | 中高 | 中 |
| **平價** | ATM | 執行價 = 股價 | 中 | 中高 |
| **價外** | OTM | 無內含價值,純時間價值 | 低 | 高 |
| **深度價外** | Deep OTM | 極低機率履約 | 極低 | 極高 |

**大額單解讀:**

```

大額 ATM/ITM Call:
→ 機構認真看多,願意支付高權利金
→ 🟢 強烈看多訊號

大額 OTM Call:
→ 短線投機或重大事件預期
→ 🟡 投機性強

大額 Deep ITM Call:
→ 可能是合成股票策略(Synthetic Long)
→ 🟢 長期看多

```

## Unusual Options Activity(異常選擇權活動)

### 定義與篩選條件

**什麼是 Unusual Activity?**
明顯異於正常交易模式的大額單。

**篩選條件:**

| 條件 | 閾值 | 含義 |
|------|------|------|
| **交易量(Volume)** | > 平均量 5倍 | 異常活躍 |
| **未平倉量(OI)比** | Volume/OI > 2 | 大量新倉位 |
| **單筆交易金額** | > $500K | 機構級別交易 |
| **買方主動** | 價格接近 Ask | 急迫性 |

**範例:**

```

NVDA 600 Call (30 DTE)
正常日均量: 500 口
今日成交: 5,000 口(10倍!)
未平倉量: 2,000 口
Volume/OI = 2.5(大量新倉位)
→ Unusual Activity 🔔

```

### Golden Sweep(黃金掃單)

**定義:**
單筆超大額選擇權交易,通常由機構執行的分批買進。

**特徵:**
```

✅ 單筆金額 > \$1M
✅ 買方主動(接近 Ask)
✅ 分批快速執行(數秒內)
✅ 橫跨多個執行價
→ 機構積極建倉,極強訊號

```

**範例:**

```

TSLA Golden Sweep:

- 700 Call x 1,000口 @ \$8.50 (850萬美元)
- 710 Call x 800口 @ \$7.20 (576萬美元)
- 720 Call x 600口 @ \$6.00 (360萬美元)
→ 總金額 1,786萬美元
→ 3分鐘內完成
→ 機構重注看多 🔥🔥🔥

```

**追蹤平台:**
- FlowAlgo
- Cheddar Flow  
- BlackBoxStocks

## Options Flow 解讀策略

### 策略 1: 跟隨大額看多單

**條件:**
```

1. 大額 Call(> \$500K)
2. 買方主動(接近 Ask)
3. 到期日 30-90 DTE(中期)
4. 執行價 ATM 或 ITM(認真看多)
5. Volume/OI > 2(大量新倉位)
→ 機構看多,跟進
```

**執行:**
- 買進正股或 Call 選擇權
- 配置 30-40% 倉位
- 停利: 漲幅 > 15% 或選擇權到期前 2 週
- 停損: -8%

**歷史勝率:** 約 65-70%

**案例:**

**NVIDIA (NVDA) - 2023年5月:**
```

5/18 出現大額 Golden Sweep:

- 420 Call (60 DTE) x 2,000口
- 總金額 \$12M
- 買方主動

股價當時: \$385
結果: 45天後股價 \$470 (+22%)

```

### 策略 2: 大額 Put 避險

**條件:**
```

1. 大額 Put(> \$1M)
2. 執行價略低於當前股價(5-10%)
3. 到期日 60-180 DTE(中長期)
4. 機構級別交易(> 1,000口)
→ 機構避險,市場可能下跌
```

**解讀:**
- 機構買 Put ≠ 一定看空
- 可能是對沖多頭部位
- 但大量買 Put = 預期波動增加

**執行:**
- 🔴 減碼持股 20-30%
- 或買入 Put 保護
- 提高現金比例

**案例:**

**SPY - 2022年1月:**
```

1/5 出現巨額 Put:

- SPY 450 Put (90 DTE) x 10,000口
- 總金額 \$25M

SPY 當時: \$475
結果: 3個月後 SPY 跌至 \$410(-14%)

```

### 策略 3: 財報前 Options Flow

**原理:**
內部人或消息靈通者在財報前布局。

**看多訊號(買進股票):**
```

財報前 3-7 天:
✅ 大額 ATM Call
✅ 到期日涵蓋財報(但不緊貼)
✅ 多筆連續大額單
→ 預期財報利多

```

**看空訊號(避開):**
```

財報前 3-7 天:
🔴 大額 ATM Put
🔴 到期日涵蓋財報
🔴 持續湧入 Put
→ 預期財報利空

```

**歷史勝率:** 約 60-65%

**案例:**

**Meta (META) - 2023年10月:**
```

財報前 4 天:

- 320 Call (涵蓋財報) 大額湧入
- 連續 3 天 Volume/OI > 3

財報結果: EPS 與營收雙超預期
股價: \$310 → \$360 (+16%)

```

## Options Flow 進階技巧

### 1. Dark Pool + Options Flow 配合

**Dark Pool:**
機構私下交易股票的場所,不在公開市場顯示。

**配合判斷:**

| Dark Pool | Options Flow | 解讀 | 操作 |
|-----------|-------------|------|------|
| **大量買進** | 大額 Call | 機構高度看多 | 🟢 強烈買進 |
| **大量賣出** | 大額 Put | 機構出貨 | 🔴 立即退出 |
| **大量買進** | 大額 Put | 機構買股+避險 | 🟡 謹慎看多 |
| **大量賣出** | 大額 Call | 機構賣股收權利金 | 🔴 看空 |

**最強組合:**
```

Dark Pool 大量買進 + 大額 Call Golden Sweep
→ 機構全方位做多
→ 🔥🔥🔥 最強看多訊號

```

### 2. Options Flow 反轉訊號

**由多轉空:**
```

連續數天 Call 湧入
→ 突然出現巨額 Put Golden Sweep
→ 🔴 機構轉向看空,立即警戒

```

**由空轉多:**
```

連續數天 Put 湧入
→ 突然出現巨額 Call Golden Sweep
→ 🟢 機構轉向看多,可能見底

```

### 3. Smart Money vs Dumb Money

**Smart Money(聰明錢):**
- 買 ITM/ATM 選擇權(認真看多/空)
- 中長期到期日(30-180 DTE)
- 大額單(> $500K)

**Dumb Money(散戶錢):**
- 買 OTM 選擇權(賭博心態)
- 短期到期日(0-7 DTE)
- 小額多筆(< $10K)

**反向指標:**
```

散戶瘋狂買 OTM Call(0 DTE)
→ 🔴 市場頂部訊號,考慮賣出

散戶瘋狂買 OTM Put
→ 🟢 市場底部訊號,考慮買進

```

## Options Flow 陷阱與誤區

### 誤區 1: 所有大額單都要跟

**錯誤觀念:**
只要看到大額選擇權單就跟進。

**事實:**
- 可能是機構平倉(STC/BTC)
- 可能是避險(不代表看多/空)
- 需配合其他指標確認

**判斷方法:**
```

確認是 BTO(Buy to Open)才跟
確認 Volume/OI > 2(新倉位)
確認買方主動(@ Ask)

```

### 誤區 2: 短期選擇權最準

**錯誤觀念:**
0-7 DTE 選擇權最即時,最準確。

**事實:**
- 短期選擇權多是散戶賭博
- 機構做長期布局(30-180 DTE)
- 長期選擇權參考價值更高

### 誤區 3: Call 多 = 一定漲

**錯誤觀念:**
看到 Call 湧入就買股票。

**事實:**
- 可能是 Sell Call(收權利金,看空)
- 需確認是 Buy Call 且買方主動
- 配合 Dark Pool 與技術面

## Options Flow 檢查清單

### 強烈看多訊號確認清單

```

□ 大額 Call(> \$500K)
□ Buy to Open(BTO)
□ 買方主動(@ Ask)
□ 執行價 ATM 或 ITM
□ 到期日 30-90 DTE
□ Volume/OI > 2
□ 多筆連續大額單(Golden Sweep)
□ Dark Pool 同步買進(若有數據)
□ 技術面突破關鍵阻力
→ 強烈看多,可跟進

```

### 強烈看空訊號確認清單

```

□ 大額 Put(> \$1M)
□ Buy to Open(BTO)
□ 買方主動(@ Ask)
□ 執行價略低於股價(5-10%)
□ 到期日 60-180 DTE
□ Volume/OI > 2
□ 連續數日 Put 持續湧入
□ Dark Pool 同步賣出(若有數據)
□ 技術面跌破關鍵支撐
→ 強烈看空,減碼或避險

```

### 財報前 Flow 確認清單

```

□ 財報前 3-7 天出現大額單
□ 到期日涵蓋財報
□ ATM 執行價
□ 連續 2-3 天同方向湧入
□ 總金額 > \$5M
□ 基本面與技術面配合
→ 可參考 Flow 方向布局

```

## 結論

Options Flow 是窺探機構動向的最佳工具,但需要經驗與技巧才能正確解讀。

**核心原則:**
1. ✅ **大額 Call(> $500K) + 買方主動 = 強烈看多**
2. ✅ **Golden Sweep = 機構重注,最強訊號**
3. ✅ **長期選擇權(30-180 DTE)比短期更可靠**
4. ✅ **配合 Dark Pool 與技術面確認**
5. ✅ **財報前 Flow 勝率約 60-65%**

**優先順序:**
```

Golden Sweep > 大額 ITM/ATM > 大額 OTM
長期(> 30 DTE) > 短期(< 7 DTE)
買方主動 > 賣方主動
新倉位(Vol/OI > 2) > 平倉

```

**最佳實踐:**
- 使用付費工具追蹤 Golden Sweep(值得投資)
- 只跟隨「買方主動 + BTO + 大額」的單
- 不盲目跟單,配合基本面與技術面
- 設定嚴格停損(-8~10%)

Options Flow 像追蹤大白鯊的遷徙,跟對了吃香喝辣,跟錯了被當點心。保持警惕,理性判斷!
