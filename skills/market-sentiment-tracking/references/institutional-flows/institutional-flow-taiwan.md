# 台股法人買賣超分析方法

## 概述

台股三大法人(外資、投信、自營商)的買賣超數據是市場最重要的資金流向指標。法人擁有龐大資金與專業研究團隊,其動向往往領先散戶,是判斷市場趨勢的關鍵。本文檔詳細說明如何解讀與應用三大法人數據。

---

## 三大法人基本介紹

### 法人特性比較

| 法人類型 | 資金規模 | 投資風格 | 影響力 | 參考價值 |
|---------|---------|---------|-------|---------|
| **外資** | 最大(兆級) | 中長期趨勢 | ⭐⭐⭐⭐⭐ | 最高(方向領先指標) |
| **投信** | 中等(百億) | 中期波段 | ⭐⭐⭐ | 高(籌碼集中指標) |
| **自營商** | 較小 | 短線避險 | ⭐⭐ | 中(短線參考) |

### 1. 外資(Foreign Institutional Investors)

**特性:**
- 資金規模最大,持股佔比約 40%
- 掌握全球資訊優勢
- 投資決策理性,較少情緒化
- 操作週期:中長期(數週至數月)

**影響力:**
- 🔴 連續賣超 > 5 天 = 趨勢轉弱信號
- 🟢 連續買超 > 5 天 = 趨勢轉強信號
- 單日買賣超 > 200億 = 重大方向改變

**典型行為:**
- 趨勢確立後持續操作(不輕易反轉)
- 大盤轉折點通常由外資主導
- 偏好大型權值股(台積電、鴻海等)

### 2. 投信(Domestic Investment Trust)

**特性:**
- 資金規模中等(管理共同基金、ETF)
- 受法規限制,持股集中度有上限
- 操作較為穩健,重視基本面
- 操作週期:中期(數週至數月)

**影響力:**
- 🟢 連續買超 = 籌碼集中,不易被洗出
- 🔴 連續賣超 = 籌碼鬆動,壓力增加
- 持股集中度高 = 支撐力強

**典型行為:**
- 偏好中大型績優股
- 作帳行情(季末、年末)特別活躍
- 買進後較少短線進出

### 3. 自營商(Proprietary Trading)

**特性:**
- 資金規模最小
- 主要從事避險與套利
- 操作靈活,頻繁進出
- 操作週期:短期(當日至數週)

**影響力:**
- ⭐ 參考價值最低
- 主要用於避險,非趨勢判斷
- 期現貨套利為主

**典型行為:**
- 買現貨、賣期貨(或反向)
- 不代表看多或看空
- 對大盤方向參考意義較小

## 法人買賣超數據解讀

### 1. 單日買賣超

**定義:**
```

買賣超金額 = 買進金額 - 賣出金額
正值 = 淨買進(買超)
負值 = 淨賣出(賣超)

```

**外資單日買賣超判斷:**

| 單日金額 | 力道 | 市場反應 | 操作建議 |
|---------|------|---------|---------|
| **> +200億** | 極強買盤 | 通常大漲 | 🟢 積極做多 |
| **+100~200億** | 強買盤 | 偏多 | 🟢 順勢做多 |
| **+50~100億** | 中度買盤 | 持穩 | 🟡 持有 |
| **-50~+50億** | 中性 | 盤整 | 🟡 觀望 |
| **-100~-50億** | 中度賣壓 | 偏弱 | 🔴 謹慎 |
| **-200~-100億** | 強賣壓 | 通常下跌 | 🔴 減碼 |
| **< -200億** | 極強賣壓 | 通常大跌 | 🔴 避險 |

**投信單日買賣超判斷:**

| 單日金額 | 力道 | 含義 |
|---------|------|------|
| **> +30億** | 極強買盤 | 積極布局 |
| **+10~30億** | 強買盤 | 正常買進 |
| **-10~+10億** | 中性 | 正常波動 |
| **< -30億** | 強賣壓 | 大幅調節 |

### 2. 連續買賣超(最重要!)

**外資連續買賣超判斷:**

| 連續天數 | 方向 | 趨勢強度 | 操作建議 |
|---------|------|---------|---------|
| **連買 10天+** | 買超 | 強勢多頭 | 🟢 持續持有,不輕易賣出 |
| **連買 5-10天** | 買超 | 多頭趨勢 | 🟢 順勢做多 |
| **連買 3-5天** | 買超 | 偏多 | 🟡 觀察確認 |
| **連賣 3-5天** | 賣超 | 偏弱 | 🟡 提高警覺 |
| **連賣 5-10天** | 賣超 | 空頭趨勢 | 🔴 減碼避險 |
| **連賣 10天+** | 賣超 | 強勢空頭 | 🔴 大幅減碼或退出 |

**歷史統計(2015-2024):**
- 外資連買 > 10 天,後續 1 個月漲機率 78%
- 外資連賣 > 10 天,後續 1 個月跌機率 72%

**投信連續買賣超判斷:**

| 連續天數 | 方向 | 含義 | 操作建議 |
|---------|------|------|---------|
| **連買 5天+** | 買超 | 籌碼集中 | 🟢 支撐力強 |
| **連買 10天+** | 買超 | 高度看好 | 🟢 強力支撐 |
| **連賣 5天+** | 賣超 | 籌碼鬆動 | 🔴 壓力增加 |

### 3. 累積買賣超

**定義:**
```

近 N 日累積買賣超 = Σ(過去 N 日單日買賣超)

```

**外資累積買賣超(20日)判斷:**

| 累積金額 | 趨勢 | 操作建議 |
|---------|------|---------|
| **> +1,000億** | 強勢多頭 | 🟢 積極做多 |
| **+500~1,000億** | 多頭 | 🟢 持有 |
| **-500~+500億** | 盤整 | 🟡 中性 |
| **-1,000~-500億** | 空頭 | 🔴 減碼 |
| **< -1,000億** | 強勢空頭 | 🔴 避險 |

## 法人動向進階分析

### 1. 外資持股比例變化

**定義:**
```

外資持股比例 = 外資持有股數 / 總流通股數 × 100%

```

**解讀標準:**

| 持股比例變化 | 趨勢 | 操作建議 |
|------------|------|---------|
| **持續上升** | 外資加碼 | 🟢 看好後市 |
| **持續下降** | 外資減碼 | 🔴 看淡後市 |
| **持股 > 30%** | 高度控盤 | 🟢 外資影響力大 |
| **持股 < 10%** | 低度參與 | 🟡 外資興趣低 |

**個股應用:**

**範例 1: 台積電 2330**
```

2023年初: 外資持股 76%
2023年底: 外資持股 79% (+3%)
→ 持續加碼,看好長期
→ 建議: 跟隨外資,長期持有

```

**範例 2: 某電子股**
```

2022年初: 外資持股 35%
2022年底: 外資持股 22% (-13%)
→ 大幅減碼,看淡後市
→ 建議: 避開該標的

```

### 2. 投信持股集中度

**前十大投信持股比例:**

| 集中度 | 含義 | 支撐力 | 操作建議 |
|-------|------|-------|---------|
| **> 15%** | 極度集中 | 極強 | 🟢 不易被洗出 |
| **10-15%** | 高度集中 | 強 | 🟢 支撐力強 |
| **5-10%** | 中度集中 | 中 | 🟡 正常 |
| **< 5%** | 低度集中 | 弱 | 🔴 支撐力弱 |

**集中度上升 = 看好,集中度下降 = 看淡**

### 3. 法人同步性分析

**三大法人同步買超(最強信號):**

```

外資買超 + 投信買超 + 自營買超
→ 三大法人一致看多
→ 🟢 最強買進信號

```

**歷史勝率:** 三法人同步買超,次日上漲機率 > 75%

**三大法人同步賣超(最弱信號):**

```

外資賣超 + 投信賣超 + 自營賣超
→ 三大法人一致看空
→ 🔴 最強賣出信號

```

**歷史勝率:** 三法人同步賣超,次日下跌機率 > 70%

**法人分歧(需觀望):**

```

外資買超 + 投信賣超
或
外資賣超 + 投信買超
→ 法人意見分歧
→ 🟡 觀望,等待一致

```

## 法人買賣超實戰策略

### 策略 1: 外資連續買超順勢策略

**買進條件:**
```

1. 外資連續買超 > 5 天
2. 累積買超 > 200億
3. 加權指數突破 20日均線
4. 成交量放大
→ 趨勢確立,順勢做多
```

**執行:**
- 進場買進大盤 ETF 或權值股
- 配置 50-70% 倉位
- 停利: 外資轉為連續賣超 3 天
- 停損: 跌破進場價 -5%

**歷史案例:**

**案例 1: 2023年11月-12月 AI行情**
```

2023/11/15 開始: 外資連續買超 15 天

- 累積買超: 1,250億
- 加權指數: 16,500 → 17,800 (+7.9%)
策略: 順勢做多,獲利豐厚

```

**案例 2: 2020年4月 疫後反彈**
```

2020/4/6 開始: 外資連續買超 20 天

- 累積買超: 2,100億
- 加權指數: 10,000 → 11,500 (+15%)
策略: 跟隨外資,大幅獲利

```

### 策略 2: 外資連續賣超避險策略

**賣出條件:**
```

1. 外資連續賣超 > 5 天
2. 累積賣超 > 200億
3. 加權指數跌破 20日均線
4. 成交量萎縮
→ 趨勢轉弱,減碼避險
```

**執行:**
- 減碼 30-50%
- 增加現金比例
- 避開權值股(外資重倉)
- 再進場條件: 外資轉為連續買超

**歷史案例:**

**案例 1: 2022年5月 通膨恐慌**
```

2022/5/10 開始: 外資連續賣超 12 天

- 累積賣超: -850億
- 加權指數: 16,800 → 15,200 (-9.5%)
策略: 及時減碼,避開重挫

```

### 策略 3: 投信作帳行情

**原理:**
投信在季末(3/6/9/12月底)與年末會美化帳面,拉抬持股。

**判斷條件:**
```

1. 季末前 2 週
2. 投信連續買超特定個股
3. 該股投信持股集中度 > 10%
→ 可能是作帳標的
```

**操作策略:**
- 🟢 季末前 2 週買進
- 🔴 季末最後 2 天賣出(作帳結束)
- 風險: 若行情不佳,作帳可能不明顯

**歷史勝率:** 約 60-65%

## 法人與散戶對比分析

### 法人 vs 散戶籌碼健康度

**健康多頭(法人主導):**

```

✅ 外資持續買超
✅ 投信持股集中度上升
✅ 融資餘額減少(散戶賣出)
✅ 券資比上升
✅ 股價穩定上漲
→ 法人吃貨,散戶賣出,籌碼健康

```

**不健康多頭(散戶追高):**

```

🔴 外資持續賣超
🔴 投信持股集中度下降
🔴 融資餘額暴增(散戶追買)
🔴 券資比下降
🔴 股價急漲
→ 法人出貨,散戶追高,危險!

```

### 法人與散戶背離策略

**外資買 + 散戶賣(最佳情境):**

```

外資買超 + 融資減少
→ 法人吃貨,散戶賣出
→ 🟢 最健康的上漲
→ 建議: 跟隨法人,積極持有

```

**外資賣 + 散戶買(最差情境):**

```

外資賣超 + 融資增加
→ 法人出貨,散戶接手
→ 🔴 最危險的上漲
→ 建議: 立即退出

```

## 個股法人分析實戰

### 實戰案例 1: 台積電(2330)

**2023年10月-12月 AI行情:**

| 日期 | 股價 | 外資買賣超(億) | 投信買賣超(億) | 累積(外) |
|------|------|--------------|-------------|---------|
| 10/2 | 520 | +15 | +2 | +15 |
| 10/9 | 540 | +22 | +3 | +152 |
| 10/16 | 560 | +28 | +5 | +318 |
| 10/23 | 580 | +35 | +4 | +563 |
| 10/30 | 600 | +30 | +6 | +773 |

**分析:**
- 外資連續買超 > 20 天
- 累積買超 > 700億
- 投信同步買超
- 股價穩定上漲
- → 典型法人主導行情,跟隨做多 🟢

**結果:** 股價從 520 漲至 650 (+25%)

### 實戰案例 2: 某航運股(長榮 2603)

**2021年5月-6月 散戶狂熱:**

| 日期 | 股價 | 外資買賣超(億) | 融資餘額(億) | 判斷 |
|------|------|--------------|------------|------|
| 5/10 | 120 | +5 | 85 | 正常 |
| 5/17 | 150 | -8 | 125 | 外資賣,散戶買 🔴 |
| 5/24 | 190 | -15 | 165 | 法人出貨 🔴 |
| 5/31 | 230 | -22 | 210 | 散戶瘋狂 🔴 |
| 6/7 | 180 | -18 | 185 | 崩跌 |

**分析:**
- 外資持續賣超
- 散戶融資暴增(從 85億 → 210億)
- 典型「法人出貨,散戶接手」
- → 極度危險,應避開 🔴

**結果:** 從 230 崩跌至 100 (-57%)

### 實戰案例 3: 鴻海(2317)作帳行情

**2022年12月 投信作帳:**

| 日期 | 股價 | 投信買賣超(億) | 投信持股集中度 |
|------|------|--------------|--------------|
| 12/12 | 96 | +3 | 8.5% |
| 12/19 | 100 | +5 | 9.2% |
| 12/26 | 105 | +8 | 10.5% |
| 12/29 | 110 | +6 | 11.2% |
| 1/3 | 106 | -4 | 11.0% |

**分析:**
- 12月投信持續買超(作帳)
- 持股集中度上升
- 股價跟隨上漲
- 年末後買盤消失,股價回落
- → 典型作帳行情 🟡

**策略:** 12月中買進,年末賣出,獲利 +10%

## 法人數據陷阱與誤區

### 誤區 1: 外資買超 = 一定漲

**錯誤觀念:**
只要外資買超就跟進買。

**事實:**
需看「連續性」與「金額」:
- 單日小額買超(< 50億)參考價值低
- 需連續買超 > 3-5 天才確認趨勢
- 配合大盤與個股技術面

### 誤區 2: 自營商數據很重要

**錯誤觀念:**
三大法人都要看。

**事實:**
- 自營商主要做避險套利
- 買賣方向與趨勢無關
- 參考價值最低,可忽略

### 誤區 3: 法人永遠是對的

**錯誤觀念:**
跟著法人做一定賺錢。

**事實:**
- 法人也會犯錯(如2000科技泡沫)
- 法人持股成本可能比你高
- 應配合基本面與估值判斷

## 法人數據實戰檢查清單

### 大盤順勢做多檢查清單

```

□ 外資連續買超 > 5 天
□ 累積買超 > 200億
□ 投信同步買超
□ 加權指數突破 20日均線
□ 融資餘額減少或持平
□ 成交量放大
□ 基本面無重大利空
→ 順勢做多,配置 50-70%

```

### 大盤減碼避險檢查清單

```

□ 外資連續賣超 > 5 天
□ 累積賣超 > 200億
□ 投信同步賣超
□ 加權指數跌破 20日均線
□ 融資餘額增加(散戶接刀)
□ 成交量萎縮
→ 減碼避險,降至 30-50%

```

### 個股健康上漲確認清單

```

□ 外資持續買超
□ 投信持股集中度上升
□ 融資餘額減少(散戶賣)
□ 券資比上升
□ 股價穩定上漲
□ 基本面良好
→ 法人主導,健康行情

```

### 個股危險訊號檢查清單

```

□ 外資持續賣超
□ 投信持股集中度下降
□ 融資餘額暴增(散戶追)
□ 券資比下降
□ 股價急漲
□ 社群瘋狂討論
→ 散戶追高,法人出貨,危險!

```

## 結論

法人買賣超數據是台股最重要的籌碼指標之一,尤其外資動向具有極高參考價值。

**核心原則:**
1. ✅ **外資連續買超 > 5天 = 順勢做多**
2. ✅ **外資連續賣超 > 5天 = 減碼避險**
3. ✅ **三法人同步買超 = 最強信號**
4. ✅ **外資買 + 散戶賣 = 最健康上漲**
5. ✅ **外資賣 + 散戶買 = 最危險上漲**

**優先順序:**
```

外資 > 投信 > 自營商
連續性 > 單日金額
累積買賣超 > 單日買賣超

```

**最佳實踐:**
- 外資是最重要指標(持股佔比 40%,影響力最大)
- 看「連續性」而非單日波動
- 配合散戶籌碼(融資融券)反向驗證
- 法人數據應與基本面、估值綜合判斷

掌握法人動向,跟著大戶走,是台股投資致勝的關鍵!
