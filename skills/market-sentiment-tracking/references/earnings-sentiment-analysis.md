# Earnings Sentiment 財報季情緒分析

## 概述

財報季是股票最大的短期催化劑,往往在數小時內決定股價漲跌 ±10-20%。Earnings Sentiment 透過分析財報前後的市場預期、分析師評級、選擇權流向等數據,預判財報結果與市場反應。本文檔詳細說明如何在財報季獲利。

---

## 財報季基礎知識

### 財報時間表

**美股財報季(Earnings Season):**

| 季度 | 財報季時間 | 涵蓋業績期間 |
|------|-----------|------------|
| **Q1** | 4月中-5月中 | 1-3月 |
| **Q2** | 7月中-8月中 | 4-6月 |
| **Q3** | 10月中-11月中 | 7-9月 |
| **Q4** | 1月中-2月中 | 10-12月 |

**台股財報時間:**

| 類型 | 公布時間 | 涵蓋期間 |
|------|---------|---------|
| **月營收** | 次月10日前 | 單月營收 |
| **季報** | 季後45日內 | 單季財報 |
| **年報** | 次年3月底前 | 全年財報 |

### 財報關鍵指標

**美股財報核心數據:**

| 指標 | 英文 | 重要性 | 市場反應 |
|------|------|--------|---------|
| **每股盈餘** | EPS | ⭐⭐⭐⭐⭐ | 超預期 → 大漲 |
| **營收** | Revenue | ⭐⭐⭐⭐⭐ | 超預期 → 上漲 |
| **毛利率** | Gross Margin | ⭐⭐⭐⭐ | 改善 → 正面 |
| **指引** | Guidance | ⭐⭐⭐⭐⭐ | 上調 → 大漲 |
| **用戶數/活躍度** | DAU/MAU | ⭐⭐⭐⭐ | 增長 → 正面 |

**判斷標準:**
```

Beat(超預期) = 實際值 > 市場預期
Meet(符合預期) = 實際值 ≈ 市場預期
Miss(不如預期) = 實際值 < 市場預期

```

**最強組合:**
```

EPS Beat + Revenue Beat + Guidance 上調
→ 🚀 Triple Beat,通常大漲 +10-20%

```

**最弱組合:**
```

EPS Miss + Revenue Miss + Guidance 下調
→ 💥 Triple Miss,通常暴跌 -15-30%

```

## 財報前情緒分析

### 1. Analyst Estimates(分析師預期)

**定義:**
華爾街分析師對 EPS 與營收的預測平均值。

**查詢方式:**
- Yahoo Finance → Analysis → Earnings Estimate
- Seeking Alpha → Earnings → Estimates
- Earnings Whisper → Whisper Number

**關鍵指標:**

**EPS Estimate Trend(預期趨勢):**

| 趨勢 | 含義 | 市場反應 | 操作建議 |
|------|------|---------|---------|
| **連續上調** | 分析師看好 | 財報前股價上漲 | 🟢 看多 |
| **連續下調** | 分析師看淡 | 財報前股價下跌 | 🔴 看空 |
| **大幅上調(> 10%)** | 重大利好預期 | 強烈看多 | 🟢 積極持有 |
| **大幅下調(> 10%)** | 重大利空預期 | 強烈看空 | 🔴 避開或放空 |

**Estimate Spread(預期分歧度):**

```

分歧度 = (最高預期 - 最低預期) / 平均預期 × 100%

```

| 分歧度 | 含義 | 財報風險 | 操作建議 |
|-------|------|---------|---------|
| **< 5%** | 高度共識 | 低(結果可預期) | 🟡 中性 |
| **5-15%** | 正常分歧 | 中等 | 🟡 正常風險 |
| **> 15%** | 分歧大 | 高(可能暴漲暴跌) | 🔴 高風險,小倉位 |

### 2. Earnings Whisper Number(耳語數字)

**定義:**
非正式流傳的「真實預期」,通常高於官方預期。

**為什麼重要?**
- 官方預期常被低估(公司好超越)
- Whisper Number 反映機構真實看法
- 若公司 Beat 官方但 Miss Whisper → 仍可能下跌

**範例:**
```

AAPL Q1 財報:

- 官方預期 EPS: \$2.10
- Whisper Number: \$2.18
- 實際公布: \$2.15
→ Beat 官方(+2.4%),但 Miss Whisper(-1.4%)
→ 結果:財報後下跌 -3%

```

**操作建議:**
- 同時參考官方預期與 Whisper
- Beat Whisper = 真正超預期
- Miss Whisper = 即使 Beat 官方也可能跌

### 3. Pre-Earnings Options Flow

**原理:**
財報前的大額選擇權單揭示內部人或消息靈通者的看法。

**看多信號:**
```

財報前 3-7 天:
✅ 大額 ATM Call(> \$500K)
✅ 買方主動(@ Ask)
✅ 到期日涵蓋財報
✅ 連續數日 Call 湧入
→ 預期財報利多 🟢

```

**看空信號:**
```

財報前 3-7 天:
🔴 大額 ATM Put(> \$500K)
🔴 買方主動(@ Ask)
🔴 到期日涵蓋財報
🔴 連續數日 Put 湧入
→ 預期財報利空 🔴

```

**歷史勝率:** 約 60-65%(需配合其他指標)

### 4. Pre-Earnings Drift(財報前漂移)

**定義:**
財報前股價的異常走勢,可能洩露內部資訊。

**類型:**

**正向漂移(看多):**
```

財報前 1-2 週:

- 股價逐步上漲
- 成交量溫和放大
- 機構買進
→ 市場預期財報利多 🟢

```

**負向漂移(看空):**
```

財報前 1-2 週:

- 股價逐步下跌
- 成交量萎縮
- 機構賣出
→ 市場預期財報利空 🔴

```

**統計發現(2015-2024):**
- 財報前 5 天漲幅 > 5% → 財報 Beat 機率 72%
- 財報前 5 天跌幅 > 5% → 財報 Miss 機率 68%

## 財報後情緒分析

### 1. Immediate Reaction(盤後立即反應)

**美股財報公布時間:**
- 多數公司:盤後(After Market Close, 4PM ET後)
- 部分公司:盤前(Before Market Open, 9:30AM ET前)

**盤後反應判斷:**

| 盤後變動 | 含義 | 次日開盤預期 |
|---------|------|------------|
| **> +10%** | 大幅超預期 | 高開持續漲 |
| **+5~10%** | 明顯超預期 | 高開,可能回吐 |
| **+2~5%** | 小幅超預期 | 小幅高開 |
| **-2~+2%** | 符合預期 | 平盤震盪 |
| **-5~-2%** | 小幅不如預期 | 小幅低開 |
| **-10~-5%** | 明顯不如預期 | 低開持續跌 |
| **< -10%** | 嚴重不如預期 | 低開暴跌 |

### 2. Conference Call Sentiment(法說會情緒)

**定義:**
財報公布後的管理層電話會議,解讀業績與展望。

**關鍵觀察點:**

**管理層語氣:**

| 語氣 | 關鍵字 | 含義 | 操作 |
|------|-------|------|------|
| **樂觀** | "strong", "accelerating", "raising guidance" | 信心強 | 🟢 持有 |
| **中性** | "in line", "as expected", "maintaining" | 正常 | 🟡 觀望 |
| **謹慎** | "cautious", "headwinds", "uncertain" | 有隱憂 | 🔴 謹慎 |
| **悲觀** | "challenging", "lowering guidance", "restructuring" | 前景差 | 🔴 減碼 |

**Q&A 環節:**

**正面信號:**
- 管理層回答具體有信心
- 上調 Guidance
- 宣布回購或增加股息

**負面信號:**
- 回答閃爍其詞
- 下調 Guidance
- 提及裁員或成本壓力

**即時追蹤:**
- Seeking Alpha Live Transcripts
- AlphaSense Earnings Call
- Bloomberg Terminal

### 3. Analyst Reactions(分析師反應)

**財報後分析師動作:**

| 動作 | 含義 | 市場反應 | 操作 |
|------|------|---------|------|
| **上調評級+目標價** | 強烈看好 | 持續上漲 | 🟢 持有 |
| **維持評級+上調目標價** | 謹慎看好 | 溫和上漲 | 🟢 持有 |
| **維持評級+目標價** | 中性 | 震盪 | 🟡 觀望 |
| **下調目標價** | 看淡 | 下跌 | 🔴 減碼 |
| **下調評級** | 強烈看空 | 持續下跌 | 🔴 退出 |

**多家分析師同時動作 = 影響力更大**

**案例:**
```

NVDA 2023 Q2 財報後:

- 15家分析師上調目標價
- 平均目標價從 \$450 → \$550
→ 市場信心增強
→ 股價持續上漲

```

### 4. Post-Earnings Announcement Drift(PEAD)

**定義:**
財報後股價的持續趨勢,可延續數週。

**統計發現(學術研究):**

**財報超預期:**
```

盤後大漲 > 5%
→ 後續 20 個交易日平均再漲 +3-5%
→ 動能延續 🟢

```

**財報不如預期:**
```

盤後大跌 > 5%
→ 後續 20 個交易日平均再跌 -3-5%
→ 賣壓延續 🔴

```

**操作策略:**
- 財報超預期大漲 → 持有 2-4 週捕捉 PEAD
- 財報不如預期大跌 → 避開 2-4 週或放空

## 財報交易策略

### 策略 1: 高確定性 Beat & Hold

**原理:**
財報前識別高機率 Beat 的標的,買進持有過財報。

**篩選條件:**
```

1. 分析師預期連續上調(過去 30 天)
2. Whisper Number > 官方預期
3. 財報前 Options Flow 顯示大額 Call
4. 財報前股價 Pre-Earnings Drift 向上
5. 過去 4 季連續 Beat(好學生)
6. 基本面趨勢向上(營收成長 > 15%)
→ 高確定性 Beat 🟢
```

**執行:**
- 財報前 3-5 天買進
- 配置 30-40% 倉位
- 持有過財報
- 停利: 財報後漲幅 > 10% 或 PEAD 結束
- 停損: 財報後跌幅 > -8%

**歷史勝率:** 約 70-75%

**案例:**

**Microsoft (MSFT) - 2024 Q2:**
```

條件確認:
✅ 分析師預期連續上調
✅ Whisper Number \$2.78 > 官方 \$2.68
✅ 財報前 Call Flow 湧入 \$8M
✅ 財報前 5 天漲 +4%
✅ 過去 4 季全 Beat

操作:

- 財報前 3 天買進 @ \$380
- 財報結果: Triple Beat
- 盤後漲 +5%, 次日再漲 +3%
- 獲利: +8%

```

### 策略 2: 財報後動能追擊

**原理:**
財報超預期大漲後,利用 PEAD 效應追擊。

**買進條件:**
```

1. 財報 Beat(EPS + Revenue)
2. 盤後/次日開盤漲幅 > 8%
3. Guidance 上調或維持
4. 分析師集體上調評級/目標價
5. 成交量暴增(> 2倍)
→ PEAD 動能強勁 🟢
```

**執行:**
- 次日開盤買進
- 配置 20-30% 倉位(追高風險)
- 持有 2-3 週
- 停利: 漲幅 > 15% 或動能減弱
- 停損: 跌破財報日高點 -5%

**歷史勝率:** 約 65-70%

**案例:**

**NVIDIA (NVDA) - 2023 Q2:**
```

財報結果:

- EPS Beat +12%
- Revenue Beat +8%
- Guidance 大幅上調
- 盤後漲 +10%, 次日再漲 +8%

操作:

- 次日開盤買進 @ \$450
- 分析師集體上調目標價
- 持有 3 週,漲至 \$520
- 獲利: +15.6%

```

### 策略 3: 財報前避險退出

**原理:**
財報前不確定性高,先獲利了結,財報後再進場。

**賣出條件:**
```

1. 已有可觀漲幅(> 30%)
2. 財報前 1 週
3. 預期分歧度高(> 15%)
4. 股價已反映樂觀預期
5. Options IV(隱含波動率)極高(> 80%)
→ 先落袋為安 🟡
```

**執行:**
- 財報前 3-5 天賣出 50-70%
- 保留 30-50% 倉位(賭財報)
- 財報後重新評估

**優點:**
- ✅ 鎖定利潤,降低風險
- ✅ 避免財報不如預期暴跌

**缺點:**
- ⚠️ 可能錯過財報大漲

**案例:**

**Apple (AAPL) - 2024 Q1:**
```

持有成本: \$150
財報前股價: \$195(+30%)
決策: 賣出 60%, 保留 40%

財報結果: EPS Miss
盤後跌 -5%, 次日再跌 -3%
保留部分虧損 -8%

整體結果:

- 賣出 60%: +30%
- 保留 40%: +30% → +20%(-8%)
- 總獲利: +26%
→ 成功避開全額回吐

```

### 策略 4: 財報不確定性賣 Straddle

**原理(進階選擇權策略):**
財報前賣出 Straddle(同時賣 Call 與 Put),賺取高額權利金。

**條件:**
```

1. 財報前 IV 極高(> 100%)
2. 股價歷史波動 < IV 隱含波動
3. 預期財報「符合預期」(不大漲大跌)
→ 賣 Straddle 賺權利金
```

**風險:**
- 🔴 若財報意外大漲/大跌 > 10%, 虧損巨大
- 🔴 僅適合經驗豐富交易者

## 產業別財報特性

### 科技股(FAANG)

**關鍵指標:**
- 用戶數(DAU/MAU)
- 雲端業務成長
- AI 相關營收

**特性:**
- 財報前 IV 極高(50-100%)
- 財報後波動 ±5-15%
- 高度關注 Guidance

**代表股票:** AAPL, MSFT, GOOGL, META, AMZN

### 晶片股

**關鍵指標:**
- 毛利率
- 訂單能見度
- 庫存水位

**特性:**
- 產業循環性強
- 財報高度相關(台積電→NVIDIA)
- 容易集體漲跌

**代表股票:** NVDA, AMD, TSM, INTC

### 消費股

**關鍵指標:**
- 同店銷售(Comp Sales)
- 利潤率
- 庫存周轉

**特性:**
- 受景氣影響大
- 財報波動較小(±3-8%)
- 重視 Guidance

**代表股票:** NKE, SBUX, MCD, TGT

## 台股財報特性

### 月營收分析

**公布時間:** 每月 10 日前

**關鍵指標:**

| 指標 | 定義 | 判斷標準 |
|------|------|---------|
| **月增率(MoM)** | 較上月成長率 | > 10% 正面 |
| **年增率(YoY)** | 較去年同月成長率 | > 20% 強勁 |
| **累計年增率** | 今年累計 vs 去年累計 | > 15% 趨勢向上 |

**操作策略:**

**營收超預期:**
```

YoY > 30% + 連續 3 個月成長
→ 法人將調高財測
→ 🟢 買進機會

```

**營收不如預期:**
```

YoY < 0%(衰退) + 連續 2 個月
→ 法人將下修財測
→ 🔴 賣出警訊

```

### 季報關鍵

**重點觀察:**
- EPS 是否超過法人預估
- 毛利率變化(產業競爭力)
- 庫存周轉天數(去化能力)

**台股特色:**
- 財報後法人報告密集發布
- 外資目標價調整影響大
- 季報後作帳行情

## 財報季檢查清單

### 財報前看多進場清單

```

□ 分析師預期連續 30 天上調
□ Whisper Number > 官方預期 5%+
□ 財報前 Call Flow > \$500K
□ 財報前 5 天股價漲 > 3%
□ 過去 4 季至少 3 季 Beat
□ 基本面趨勢向上(營收成長 > 15%)
□ 產業景氣向上
□ Options IV < 80%(不過度昂貴)
→ 高確定性 Beat,買進持有過財報

```

### 財報後追擊動能清單

```

□ EPS Beat > 5%
□ Revenue Beat > 3%
□ Guidance 上調或維持
□ 盤後/次日漲幅 > 8%
□ 至少 3 家分析師上調評級/目標價
□ 成交量 > 平均 2 倍
□ Conference Call 語氣樂觀
□ 產業同業也 Beat(產業向上)
→ PEAD 動能強勁,追擊

```

### 財報前避險清單

```

□ 持股已有 > 30% 漲幅
□ 預期分歧度 > 15%(不確定性高)
□ Options IV > 80%(過度昂貴)
□ 財報前股價已大漲(反映樂觀預期)
□ 分析師預期停止上調
□ 產業同業財報不佳
□ 技術面出現頂部信號
→ 先獲利 50-70%,降低風險

```

## 結論

財報季是股票最大的短期催化劑,掌握 Earnings Sentiment 能大幅提高勝率。

**核心原則:**
1. ✅ **分析師預期連續上調 + Whisper Beat = 高確定性 Beat**
2. ✅ **財報前 Call Flow + Pre-Drift = 內部人看好**
3. ✅ **財報大漲 > 8% + 分析師上調 = PEAD 動能**
4. ✅ **過去連續 Beat 的「好學生」容易繼續 Beat**
5. ✅ **Guidance > EPS/Revenue(最重要!)**

**風險管理:**
- 財報前不確定性高,控制倉位 ≤ 40%
- 已有高漲幅(> 30%)考慮財報前獲利
- 分歧度高(> 15%)降低倉位或避開
- 財報後大跌立即停損,不抱僥倖

**最佳實踐:**
- 追蹤分析師預期變化(每週檢查)
- 財報前 3-7 天觀察 Options Flow
- 財報當晚立即收聽 Conference Call
- 利用 PEAD 持有 2-4 週

財報季就像考試,準備充分的學生(公司)容易考好,而市場會獎勵超預期者。做好功課,跟著好學生走!📊📈
