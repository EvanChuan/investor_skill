# 新聞情緒 NLP 分析指南

## 概述

新聞媒體報導反映市場主流情緒與預期。本指南詳細說明如何透過自然語言處理（NLP）技術分析財經新聞情緒，量化正面/負面/中性報導比例，識別情緒極端值，並預測市場情緒轉折點。

---

## 核心概念

### 新聞情緒的重要性

**為何追蹤新聞情緒？**

1. **反映市場主流看法：** 新聞媒體報導影響投資人情緒
2. **情緒極端值領先指標：** 極度正面/負面報導往往是反轉信號
3. **量化主觀感受：** 將「感覺」轉化為可追蹤的數據
4. **驗證其他指標：** 與分析師共識、社群情緒交叉驗證

**新聞情緒的影響路徑：**
```

新聞報導 → 投資人情緒變化 → 買賣決策 → 股價變動

```

**新聞情緒的滯後性：**
- 新聞往往「追逐」價格（股價漲 → 正面報導增加）
- 極端情緒時新聞滯後最嚴重（如暴漲後媒體一致看多）
- **逆向操作關鍵：** 當新聞極度樂觀時，往往是見頂信號

### 新聞情緒分類

**三分類系統：**

| 情緒分類 | 評分 | 關鍵字範例 | 標題範例 |
|---------|------|-----------|---------|
| 正面（Positive） | +1 | 突破、創新高、強勁、超預期、看好 | "Apple 業績超預期，股價創歷史新高" |
| 中性（Neutral） | 0 | 持平、觀望、整理、公布、宣布 | "Tesla 公布第三季財報" |
| 負面（Negative） | -1 | 暴跌、危機、警告、下修、虧損 | "Meta 大裁員，股價重挫 20%" |

**情緒評分計算：**
```

情緒分數 = (正面新聞數 - 負面新聞數) / 總新聞數

範圍：-1（極度負面）~ +1（極度正面）

```

## NLP 情緒分析方法

### 方法 1：關鍵字頻率分析（簡易）

**步驟：**

1. **蒐集近 7 天或 30 天新聞標題**
   - 搜尋關鍵字：公司名稱、股票代號
   - 蒐集標題與摘要（節省處理時間）

2. **建立正負面關鍵字清單**

**正面關鍵字（中文）：**
```

看好、突破、創新高、強勁、超預期、大漲、飆漲、
獲利、成長、創新、成功、利多、樂觀、買進、
轉盈、訂單、合約、上調、升評

```

**負面關鍵字（中文）：**
```

看壞、暴跌、重挫、危機、警告、下修、虧損、
裁員、倒閉、破產、衰退、下滑、失敗、利空、
悲觀、賣出、降評、違約、召回、停產

```

**正面關鍵字（英文）：**
```

bullish, surge, breakthrough, strong, beat expectations,
soar, rally, profit, growth, innovation, success,
upgrade, buy, bullish, outperform

```

**負面關鍵字（英文）：**
```

bearish, plunge, crash, crisis, warning, cut, loss,
layoff, bankruptcy, recession, decline, failure,
downgrade, sell, underperform, risk

```

3. **計算關鍵字出現頻率**
   - 正面關鍵字出現次數 vs 負面關鍵字出現次數
   - 計算比例

4. **情緒評分**
```

正面關鍵字數：25
負面關鍵字數：10
總新聞數：50

正面比例 = 25 / 50 = 50%
負面比例 = 10 / 50 = 20%
中性比例 = 1 - 50% - 20% = 30%

情緒分數 = (25 - 10) / 50 = 0.3（偏正面）

```

**優點：** 簡單快速，不需 AI 模型
**缺點：** 準確度較低，無法理解語意（如反諷）

---

### 方法 2：AI 模型情緒分類（推薦）

**步驟：**

1. **蒐集新聞標題與摘要**
   - 近 7 天或 30 天
   - 每則新聞限制在 200 字內（節省 token）

2. **使用 Claude / ChatGPT 批次分析**

**Prompt 範本：**
```

請分析以下新聞標題與摘要的情緒（正面/中性/負面）：

新聞 1：[標題與摘要]
新聞 2：[標題與摘要]
...
新聞 N：[標題與摘要]

請以以下格式回覆：
新聞 1: 正面
新聞 2: 負面
...
新聞 N: 中性

最後統計：
正面新聞數：X
中性新聞數：Y
負面新聞數：Z
情緒分數：(X - Z) / (X + Y + Z)

```

3. **批次處理**
   - 每次處理 20-30 則新聞（避免 token 超限）
   - 多批次處理後加總統計

4. **計算最終情緒分數**

**範例輸出：**
```

總新聞數：50
正面新聞：30（60%）
中性新聞：12（24%）
負面新聞：8（16%）

情緒分數 = (30 - 8) / 50 = 0.44（偏正面）

```

**優點：** 準確度高，能理解語意與上下文
**缺點：** 需要 API 成本（或手動操作）

## 情緒極端值偵測

### 極度正面情緒（🔴 警訊）

**觸發條件（需同時滿足）：**
1. 正面新聞比例 > 80%
2. 負面新聞比例 < 5%
3. 情緒分數 > 0.7
4. 持續 1 週以上

**市場特徵：**
- 媒體一致看好
- 新聞標題充滿「突破」「創新高」「史上最佳」
- 散戶 FOMO 情緒高漲

**歷史案例：**
- **Tesla (2021/11)：** 股價 $1,200，媒體 85% 正面報導 → 隨後跌至 $600（-50%）
- **Nvidia (2024/6)：** 媒體一致看好 AI → 短線回調 -20%

**逆向操作建議：**
- 🔴 考慮獲利了結（至少 30-50%）
- 🔴 不追高
- 🟡 等待情緒冷卻後再進場

### 極度負面情緒（🟢 逆向機會）

**觸發條件（需同時滿足）：**
1. 負面新聞比例 > 80%
2. 正面新聞比例 < 5%
3. 情緒分數 < -0.7
4. 持續 1 週以上

**市場特徵：**
- 媒體一致看空
- 新聞標題充滿「危機」「崩盤」「末日」
- 散戶恐慌性拋售

**歷史案例：**
- **Meta (2022/11)：** 股價 $90，媒體 90% 負面報導 → 隨後漲至 $475（+430%）
- **銀行股 (2023/3)：** 矽谷銀行倒閉，媒體恐慌報導 → 大型銀行股隨後反彈 +50%

**逆向操作建議：**
- 🟢 深度研究基本面（排除真正危機）
- 🟢 若基本面健康，逢低分批佈局
- 🟢 嚴格停損（-15%）

### 情緒-價格背離

**頂背離（🔴 警訊）：**
- 股價創新高 + 正面新聞比例不再創新高
- 股價上漲 + 負面新聞開始增加

**範例：**
```

1 月：股價 \$100，正面新聞 70%
2 月：股價 \$120，正面新聞 75%
3 月：股價 \$140，正面新聞 70%（不再創新高）
→ 頂背離警訊

```

**底背離（🟢 機會）：**
- 股價創新低 + 負面新聞比例不再創新高
- 股價下跌 + 正面新聞開始增加

**範例：**
```

1 月：股價 \$100，負面新聞 80%
2 月：股價 \$80，負面新聞 85%
3 月：股價 \$70，負面新聞 75%（不再創新高）
→ 底背離，可能見底

```

## 時間序列分析

### 情緒變化速度

**快速轉正（🟢 正面信號）：**
- 情緒分數從 -0.5 升至 +0.3（1-2 週內）
- 負面新聞快速減少
- 正面新聞開始增加

**操作建議：**
- 🟢 情緒改善，可逢低佈局
- 🟢 結合基本面與估值判斷

**快速轉負（🔴 警訊）：**
- 情緒分數從 +0.5 降至 -0.3（1-2 週內）
- 正面新聞快速減少
- 負面新聞開始增加

**操作建議：**
- 🔴 情緒惡化，考慮減碼
- 🔴 檢視基本面是否惡化

### 情緒持續時間

**短暫情緒（1-3 天）：**
- 單一事件引發（如突發新聞）
- 情緒快速恢復正常
- **操作：** 忽略短暫波動

**中期情緒（1-4 週）：**
- 持續性事件（如業績季、產品發布）
- 情緒維持一段時間
- **操作：** 關注但不過度反應

**長期情緒（數月）：**
- 結構性變化（如產業趨勢、政策變化）
- 情緒持續偏正面或負面
- **操作：** 重要信號，需深度分析

## 新聞來源可信度評估

### 可信度分級

**高可信度（A 級）：**
- Bloomberg、Reuters、Wall Street Journal
- Financial Times、The Economist
- 官方公告（SEC Filing、公司財報）

**中等可信度（B 級）：**
- CNBC、MarketWatch、Yahoo Finance
- Seeking Alpha（作者品質參差）
- 主流財經媒體

**低可信度（C 級）：**
- 小型財經網站
- 社群媒體個人發文
- 未查證的傳聞

**操作建議：**
- 優先參考 A 級來源
- B 級來源需交叉驗證
- C 級來源僅作參考（情緒指標）

## 實戰操作流程

### 每週新聞情緒監測（推薦）

**步驟 1：蒐集新聞（每週日執行）**
- 搜尋關注股票近 7 天新聞
- 蒐集標題與摘要（20-30 則）

**步驟 2：AI 情緒分析**
- 使用 Claude / ChatGPT 分析
- 統計正面/中性/負面比例
- 計算情緒分數

**步驟 3：記錄歷史數據**
- 建立 Excel / Google Sheets
- 記錄每週情緒分數
- 繪製情緒趨勢圖

**步驟 4：比對股價變化**
- 觀察情緒 vs 股價相關性
- 識別情緒-價格背離

**步驟 5：操作決策**
- 結合其他情緒指標（分析師共識、社群情緒）
- 產出操作建議

**時間成本：** 約 30 分鐘/週/股票

- 重要事件（如業績公布、產品發布）

## 常見誤區

**誤區 1：新聞情緒 = 未來股價**
- 新聞往往滯後價格（追逐價格）
- 極端情緒時新聞最不可信

**誤區 2：只看標題不看內文**
- 標題可能聳動（點擊誘餌）
- 需閱讀摘要或內文確認

**誤區 3：所有新聞來源同等重要**
- 優先參考高可信度來源
- 小型網站可能有偏誤

**誤區 4：短期情緒波動過度反應**
- 單一負面新聞不代表趨勢
- 需觀察持續性

**誤區 5：忽略情緒-價格背離**
- 背離是重要的反轉信號
- 需定期檢視

## 總結

**新聞情緒 NLP 分析核心要點：**

1. **情緒極端值：** 正面 > 80% 或負面 > 80% = 反轉信號
2. **情緒-價格背離：** 股價創新高但情緒不再創新高 = 警訊
3. **情緒變化速度：** 快速轉正/轉負 = 重要信號
4. **持續時間：** 長期情緒（數月）比短期情緒重要
5. **逆向操作：** 極度負面 + 基本面健康 = 逢低買進機會

**實戰檢查清單：**
- ✅ 每週追蹤關注股票新聞情緒
- ✅ 記錄歷史數據（至少 3 個月）
- ✅ 繪製情緒趨勢圖
- ✅ 比對股價變化（識別背離）
- ✅ 結合分析師共識與社群情緒
- ❌ 極度正面（> 80%）後追高
- ❌ 短期波動過度反應
