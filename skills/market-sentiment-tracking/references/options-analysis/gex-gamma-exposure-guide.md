# GEX（Gamma Exposure）與暴力盤分析指南

## 概述

GEX（Gamma Exposure，伽馬曝險）是衡量選擇權市場對標的價格「推動力」或「壓制力」的關鍵指標。本指南詳細說明 GEX 的原理、造市商對沖機制、如何識別暴力盤與軋空機會，並提供實戰操作策略。

---

## 核心概念

### 什麼是 Gamma？

**希臘字母（Greeks）簡介：**

| 希臘字母 | 衡量內容 | 說明 |
|---------|---------|------|
| Delta (Δ) | 價格敏感度 | 標的價格變動 $1，選擇權價格變動多少 |
| Gamma (Γ) | Delta 的變化率 | 標的價格變動 $1，Delta 變動多少 |
| Theta (Θ) | 時間價值衰減 | 每天流逝，選擇權價值減少多少 |
| Vega (V) | 波動率敏感度 | 波動率變動 1%，選擇權價格變動多少 |

**Gamma 的重要性：**
- Gamma 最高的選擇權：**價平（At-The-Money, ATM）選擇權**
- Gamma 衡量「Delta 的加速度」
- 高 Gamma = 選擇權價格對標的價格變動極度敏感

### 造市商的對沖機制

**造市商（Market Maker）的角色：**
- 提供流動性（買賣雙方報價）
- 賺取買賣價差（Bid-Ask Spread）
- **必須保持 Delta 中性**（不希望承擔方向性風險）

**造市商對沖範例：**

**情境 1：投資人買進 Call（看多）**
1. 投資人：買進 Call（Delta +0.5）
2. 造市商：賣出 Call 給投資人（Delta -0.5）
3. 造市商為保持中性：**買進 0.5 股標的股票**（Delta +0.5）
4. 總 Delta = -0.5 (Call) + 0.5 (股票) = 0（中性）

**情境 2：標的股價上漲**
1. Call 的 Delta 從 0.5 增加到 0.7（因為 Gamma 存在）
2. 造市商的 Call 部位：Delta -0.7
3. 造市商持有股票：Delta +0.5
4. 總 Delta = -0.7 + 0.5 = -0.2（不再中性）
5. 造市商必須**再買進 0.2 股**來保持中性

**關鍵洞察：**
- 當大量投資人買進 Call → 造市商必須買進股票對沖 → **推升股價**
- 當大量投資人買進 Put → 造市商必須賣出股票對沖 → **壓低股價**

## GEX 的計算與意義

### GEX 的定義

**Gamma Exposure (GEX)：**
- 所有未平倉選擇權的 Gamma 加總
- 衡量造市商需要對沖的「壓力」
- **Positive GEX：** 造市商壓制波動
- **Negative GEX：** 造市商加劇波動

### Positive GEX（正伽馬）

**特徵：**
- Call 未平倉量（OI）> Put 未平倉量
- 投資人偏多（買進大量 Call）
- 造市商持有大量股票對沖

**造市商行為：**
- 股價上漲 → Delta 增加 → 造市商**賣出股票**對沖 → 壓制漲幅
- 股價下跌 → Delta 減少 → 造市商**買進股票**對沖 → 壓制跌幅
- **結果：壓制波動，形成區間盤**

**市場特徵：**
- 低波動（VIX 下降）
- 區間震盪（上下有壓力）
- 突破不易（造市商對沖壓制）

**操作建議：**
- 🟡 區間操作（低買高賣）
- 🔴 避免追漲殺跌
- 🟡 賣出選擇權（賺取時間價值）

### Negative GEX（負伽馬）

**特徵：**
- Put 未平倉量（OI）> Call 未平倉量
- 投資人偏空（買進大量 Put 避險）
- 造市商持有大量空頭股票對沖

**造市商行為：**
- 股價上漲 → Delta 減少 → 造市商**買進股票**對沖 → **加劇漲幅**
- 股價下跌 → Delta 增加 → 造市商**賣出股票**對沖 → **加劇跌幅**
- **結果：加劇波動，形成暴力盤**

**市場特徵：**
- 高波動（VIX 上升）
- 單邊走勢（暴漲或暴跌）
- 容易突破（造市商對沖加速）

**操作建議：**
- 🟢 趨勢追蹤（順勢操作）
- 🔴 避免逆勢（容易被軋）
- 🟢 買進選擇權（捕捉大波動）

## Zero Gamma Level（零伽馬位階）

### 定義

**Zero Gamma Level (ZGL)：**
- GEX 從正轉負（或負轉正）的臨界點
- 通常是**重要支撐或壓力位**
- 突破 ZGL 後，市場性質改變

### ZGL 的重要性

**ZGL 作為支撐位（股價在 ZGL 之上）：**
- 股價 > ZGL：Positive GEX，造市商壓制波動
- 股價跌破 ZGL：轉為 Negative GEX，加劇下跌
- **ZGL 是關鍵防線**

**ZGL 作為壓力位（股價在 ZGL 之下）：**
- 股價 < ZGL：Negative GEX，高波動
- 股價突破 ZGL：轉為 Positive GEX，壓制波動
- **ZGL 是關鍵突破位**

### 歷史案例

**案例 1：S&P 500 (2020/3 疫情恐慌)**
- ZGL：2800 點
- 股價跌破 ZGL → Negative GEX → 暴跌至 2200 點
- 重新站上 ZGL → Positive GEX → 開始穩定反彈

**案例 2：Tesla (2020/2)**
- ZGL：$600
- 股價突破 ZGL → Positive GEX → 暴漲至 $900
- 高點回落至 ZGL 獲得支撐

## GEX 與暴力盤偵測

### 暴力上漲（Gamma Squeeze）

**觸發條件（需同時滿足）：**
1. **Negative GEX**（Put OI > Call OI）
2. **大量 Call 未平倉（OI）集中在特定履約價**
3. **股價接近或突破該履約價**
4. **正面催化劑**（利多消息）

**暴力上漲機制：**
```

股價上漲 → Call Delta 增加 → 造市商買進股票對沖
→ 推升股價 → 更多 Call 價內 → 造市商繼續買進
→ 正向循環 → 暴力上漲

```

**歷史案例：**
- **GameStop (2021/1)：** 
  - $115 有大量 Call OI
  - 股價突破 $115 → 暴漲至 $483（+320%，2 週內）
  
- **Tesla (2020/2)：**
  - $600 有大量 Call OI
  - 股價突破 $600 → 暴漲至 $900（+50%，1 個月內）

**操作建議：**
- 🟢 若已持有，繼續持有（設移動停利）
- 🟡 若未持有，可小額追入（風險高）
- 🟢 避免放空（容易被軋）

### 暴力下跌（Negative Gamma Dump）

**觸發條件（需同時滿足）：**
1. **Negative GEX**（Put OI > Call OI）
2. **大量 Put 未平倉（OI）集中在特定履約價**
3. **股價跌破該履約價**
4. **負面催化劑**（利空消息）

**暴力下跌機制：**
```

股價下跌 → Put Delta 增加 → 造市商賣出股票對沖
→ 壓低股價 → 更多 Put 價內 → 造市商繼續賣出
→ 負向循環 → 暴力下跌

```

**歷史案例：**
- **Meta (2022/2)：** 
  - 業績不佳 + 大量 Put OI
  - 單日暴跌 -26%
  
- **Netflix (2022/4)：**
  - 訂閱數下滑 + 大量 Put OI
  - 單日暴跌 -35%

**操作建議：**
- 🔴 立即停損（不要試圖接刀）
- 🔴 避免逆勢做多（容易加速虧損）
- 🟡 等待情緒穩定後再評估

- [Vanna](https://vanna.ai/) - $49/月，視覺化 GEX 與 Vanna

### 關鍵監控指標

**1. 整體市場 GEX（S&P 500）**
- Positive GEX > $5B：強力壓制波動（區間盤）
- Positive GEX $0-$5B：溫和壓制
- Negative GEX：高波動，暴力盤

**2. 個股 GEX**
- Positive GEX：區間操作
- Negative GEX：趨勢追蹤或避開（風險高）

**3. 關鍵履約價 OI 分布**
- 找出 Call OI 與 Put OI 最集中的履約價
- 這些履約價往往是支撐或壓力

**4. Zero Gamma Level（ZGL）**
- 每日追蹤 ZGL 位置
- 股價接近 ZGL 時提高警覺

## 實戰操作策略

### 策略 1：Positive GEX 區間操作

**適用市場環境：**
- S&P 500 Positive GEX > $3B
- VIX < 20
- 個股 Positive GEX

**操作步驟：**
1. 🟢 識別區間上下界（Call/Put OI 集中位）
2. 🟢 區間下緣買進（接近 Put OI 集中位）
3. 🟢 區間上緣賣出（接近 Call OI 集中位）
4. 🟢 重複操作

**風險控制：**
- 若突破區間上緣，立即停損（可能轉為趨勢）
- 若跌破區間下緣，立即停損（可能轉為暴跌）

**歷史成功率：** 約 60%（適合震盪市）

### 策略 2：Negative GEX 趨勢追蹤

**適用市場環境：**
- S&P 500 Negative GEX
- VIX > 25
- 個股 Negative GEX + 明確趨勢

**操作步驟（做多）：**
1. 🟢 確認上升趨勢（突破關鍵壓力）
2. 🟢 股價回測支撐後買進
3. 🟢 設定移動停利（如跌破 5 日均線出場）
4. 🔴 不逆勢（避免被軋）

**操作步驟（避險）：**
1. 🔴 若無明確趨勢，暫時離場
2. 🟡 等待 Positive GEX 恢復再進場

**風險控制：**
- 嚴格停損（-5% ~ -7%）
- 避免重倉（倉位 < 10%）

### 策略 3：ZGL 突破/跌破交易

**ZGL 突破（做多）：**
1. 🟢 股價接近 ZGL 壓力位
2. 🟢 出現正面催化劑
3. 🟢 股價放量突破 ZGL
4. 🟢 買進（設停損於 ZGL 下方）

**ZGL 跌破（避險）：**
1. 🔴 股價接近 ZGL 支撐位
2. 🔴 出現負面催化劑
3. 🔴 股價跌破 ZGL
4. 🔴 立即減碼或清倉

**歷史成功率：** 約 55%（需配合其他指標）

### 策略 4：Gamma Squeeze 捕捉

**觸發條件（需同時滿足）：**
1. Negative GEX
2. 特定履約價有大量 Call OI（> 10,000 OI）
3. 股價接近該履約價（距離 < 5%）
4. 正面催化劑

**操作步驟：**
1. 🟡 若股價尚未突破，小額試單（5% 倉位）
2. 🟢 股價突破該履約價後加碼（再 5-10% 倉位）
3. 🟢 設定移動停利（跌破 5 日均線出場）
4. 🟢 不貪心（見好就收）

**風險控制：**
- 初始倉位小（5%）
- 嚴格停損（-10%）
- 見好就收（上漲 50% 後至少賣出一半）

**歷史成功率：** 約 40%（高風險高報酬）

## GEX 與其他指標結合

### GEX + VIX

| GEX 狀態 | VIX 水平 | 市場特徵 | 操作建議 |
|---------|---------|---------|---------|
| Positive | < 15 | 極度平穩（自滿） | 🟡 謹慎（可能突然反轉） |
| Positive | 15-20 | 正常區間盤 | 🟢 區間操作 |
| Negative | 20-30 | 高波動 | 🟡 趨勢追蹤或觀望 |
| Negative | > 30 | 恐慌性波動 | 🔴 避險或等待穩定 |

### GEX + Put/Call Ratio

| GEX 狀態 | Put/Call Ratio | 解讀 | 操作建議 |
|---------|---------------|------|---------|
| Negative | > 1.2 | 極度悲觀，易暴跌 | 🔴 減碼或避險 |
| Negative | 0.7-1.2 | 中性偏空，高波動 | 🟡 謹慎操作 |
| Positive | 0.7-1.0 | 正常市場 | 🟢 正常持有 |
| Positive | < 0.7 | 偏向樂觀，壓制波動 | 🟡 謹慎追高 |

### GEX + Short Interest

| GEX 狀態 | Short Interest | 解讀 | 軋空機會 |
|---------|---------------|------|---------|
| Negative | > 20% | 高放空 + 高波動 | 🟢 軋空潛力極大 |
| Negative | 10-20% | 中度放空 | 🟡 有軋空機會 |
| Positive | > 20% | 高放空但壓制波動 | 🟡 軋空力道受限 |
| Positive | < 10% | 低放空 | ❌ 無軋空機會 |

## 常見誤區

**誤區 1：Negative GEX 一定會暴漲或暴跌**
- Negative GEX 只是「容易」暴漲或暴跌
- 仍需正面/負面催化劑觸發
- 無催化劑時可能只是高波動震盪

**誤區 2：GEX 是萬能指標**
- GEX 需配合基本面、技術面、情緒面
- 單獨使用 GEX 容易誤判

**誤區 3：所有 Call OI 集中都會 Gamma Squeeze**
- 需同時滿足：Negative GEX + 大量 Call OI + 正面催化劑
- 若 Positive GEX，造市商會壓制漲幅

**誤區 4：ZGL 是絕對支撐/壓力**
- ZGL 會隨選擇權 OI 變化而移動
- 需每日更新追蹤

## 總結

**GEX 核心要點：**

1. **Positive GEX：** 造市商壓制波動 → 區間操作
2. **Negative GEX：** 造市商加劇波動 → 趨勢追蹤或避開
3. **Zero Gamma Level：** 關鍵支撐/壓力位
4. **Gamma Squeeze：** Negative GEX + 大量 Call OI + 正面催化劑
5. **風險控制：** 嚴格停損 + 小倉位試單

**實戰檢查清單：**
- ✅ 確認市場 GEX 狀態（Positive / Negative）
- ✅ 找出關鍵履約價 Call/Put OI 集中位
- ✅ 追蹤 Zero Gamma Level
- ✅ 結合 VIX、Put/Call Ratio、Short Interest
- ✅ 等待催化劑（利多/利空消息）
- ✅ 嚴格停損與倉位控制

**下一步行動：**
- 訂閱 SpotGamma 或 SqueezeMetrics（至少免費版）
- 每日追蹤 S&P 500 GEX 與 ZGL
- 監控持倉個股的 Call/Put OI 分布
- 結合第五階段其他情緒指標綜合判斷
