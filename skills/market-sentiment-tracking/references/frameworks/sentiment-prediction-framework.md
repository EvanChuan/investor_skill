# 情緒預測與轉折點判斷框架

## 概述

市場情緒並非隨機波動，而是遵循可預測的週期性模式。本指南詳細說明如何預測情緒轉折點、判斷當前情緒歷史位階、識別情緒-價格背離，並提供基於歷史模式的情緒反轉預測方法。

---

## 核心概念

### 情緒週期理論

**市場情緒週期（完整循環）：**

```

樂觀 → 興奮 → 狂熱 → 自滿 → 焦慮 → 恐慌 → 投降 → 絕望 → 懷疑 → 希望 → 樂觀
↑                                                                      ↓
└──────────────────────────────── 完整循環 ────────────────────────────┘

```

**情緒週期各階段特徵：**

| 階段 | 情緒分數 | 散戶行為 | 機構行為 | 股價 | 最佳策略 |
|------|---------|---------|---------|------|---------|
| 絕望 | 0-10 | 投降、賣出 | 逢低買進 | 底部 | 🟢 **積極買進** |
| 懷疑 | 10-25 | 觀望、不信 | 持續買進 | 初升段 | 🟢 買進 |
| 希望 | 25-40 | 小心試單 | 增持 | 穩定上漲 | 🟢 買進 |
| 樂觀 | 40-60 | 持續持有 | 持有 | 持續上漲 | 🟡 持有 |
| 興奮 | 60-75 | 加碼買進 | 開始減持 | 加速上漲 | 🟡 部分獲利 |
| 狂熱 | 75-90 | FOMO、加槓桿 | 大量減持 | 暴漲 | 🔴 **大幅減碼** |
| 自滿 | 90-100 | 全倉、勸親友 | 清倉 | 頂部 | 🔴 **清倉** |
| 焦慮 | 80-90 | 拒絕相信 | 空手 | 開始下跌 | 🔴 減碼 |
| 恐慌 | 60-80 | 恐慌賣出 | 等待機會 | 暴跌 | 🟡 觀望 |
| 投降 | 40-60 | 認賠殺出 | 準備進場 | 持續下跌 | 🟡 謹慎試單 |

**關鍵洞察：**
- 情緒極端時（< 10 或 > 90）往往是最佳進場/出場時機
- 機構與散戶行為相反（逆向操作）
- 情緒週期完整循環約 6-18 個月（視市場環境）

---

### 情緒動能與加速度

**情緒動能（Sentiment Momentum）：**
```

情緒動能 = 當前情緒分數 - 上期情緒分數

```

**情緒加速度（Sentiment Acceleration）：**
```

情緒加速度 = 當前情緒動能 - 上期情緒動能

```

**動能與加速度解讀：**

| 動能 | 加速度 | 市場狀態 | 預測 |
|------|-------|---------|------|
| 正值 | 正值 | 情緒加速上升 | 繼續上漲，但接近過熱 |
| 正值 | 負值 | 情緒上升但放緩 | 🔴 可能見頂（背離） |
| 負值 | 負值 | 情緒加速下降 | 繼續下跌，但接近恐慌 |
| 負值 | 正值 | 情緒下降但放緩 | 🟢 可能見底（背離） |

**範例計算：**
```

週 1：情緒分數 60，動能 = N/A
週 2：情緒分數 70，動能 = 70 - 60 = +10
週 3：情緒分數 75，動能 = 75 - 70 = +5，加速度 = +5 - (+10) = -5
週 4：情緒分數 78，動能 = 78 - 75 = +3，加速度 = +3 - (+5) = -2

解讀：情緒仍在上升（動能為正），但上升速度放緩（加速度為負）
→ 可能接近頂部，謹慎追高

```

## 情緒歷史分位數分析

### 計算方法

**步驟 1：蒐集歷史情緒數據**
- 至少需要 1 年歷史數據（建議 3-5 年）
- 每週記錄綜合情緒分數（0-100）

**步驟 2：計算分位數**
```

當前情緒分數在歷史數據中的排名位置

範例：

- 歷史 156 週（3 年）情緒數據
- 當前情緒分數：85
- 有 140 週的情緒分數 < 85
- 分位數 = 140 / 156 = 89.7%

解讀：當前情緒處於歷史 89.7% 分位（接近歷史高點）

```

### 分位數解讀標準

| 歷史分位數 | 市場解讀 | 操作建議 |
|----------|---------|---------|
| < 5% | 極度恐慌（歷史罕見） | 🟢 **強烈買進信號** |
| 5-10% | 極度悲觀 | 🟢 積極逢低買進 |
| 10-25% | 偏向悲觀 | 🟢 逢低佈局 |
| 25-50% | 中性偏空 | 🟡 謹慎樂觀 |
| 50-75% | 中性偏多 | 🟡 正常持有 |
| 75-90% | 偏向樂觀 | 🔴 謹慎追高 |
| 90-95% | 極度樂觀 | 🔴 考慮大幅減碼 |
| > 95% | 極度狂熱（歷史罕見） | 🔴 **強烈賣出信號** |

**歷史極端值參考：**
- 過去 3 年最低情緒分數（約 5% 分位） = 強支撐
- 過去 3 年最高情緒分數（約 95% 分位） = 強壓力

## 情緒-價格背離偵測

### 頂背離（Top Divergence）

**定義：**
- 價格創新高，但情緒分數未創新高
- 或：價格上漲，但情緒分數下降

**形成原因：**
- 機構獲利了結（不再樂觀）
- 散戶追高力道減弱
- 媒體與分析師轉向謹慎

**偵測方法：**

1. **記錄價格與情緒高點**
```

第一波高點：

- 股價：\$100
- 情緒分數：85
- 日期：2026/1/15

第二波高點：

- 股價：\$110（創新高）
- 情緒分數：80（未創新高）
- 日期：2026/2/15

→ 頂背離確認

```

2. **連續性驗證**
- 至少連續 2 週背離
- 背離幅度 > 5 分（顯著）

**頂背離操作策略：**
- 🔴 **立即獲利了結 30-50%**
- 🔴 設定嚴格停利（跌破 10 日均線出場）
- 🔴 不再加碼

**歷史成功率：** 約 65%（中期頂部預測）

---

### 底背離（Bottom Divergence）

**定義：**
- 價格創新低，但情緒分數未創新低
- 或：價格下跌，但情緒分數上升

**形成原因：**
- 機構逢低佈局（開始樂觀）
- 散戶恐慌殺盤已接近尾聲
- 媒體與分析師開始轉向正面

**偵測方法：**

1. **記錄價格與情緒低點**
```

第一波低點：

- 股價：\$100
- 情緒分數：25
- 日期：2026/1/15

第二波低點：

- 股價：\$90（創新低）
- 情緒分數：30（未創新低）
- 日期：2026/2/15

→ 底背離確認

```

2. **連續性驗證**
- 至少連續 2 週背離
- 背離幅度 > 5 分（顯著）

**底背離操作策略：**
- 🟢 **謹慎試單（5% 倉位）**
- 🟢 等待情緒穩定後加碼
- 🟢 嚴格停損（-10%）

**歷史成功率：** 約 55%（中期底部預測）

---

### 背離強度量化

**背離強度計算：**
```

背離強度 = |情緒分數變化%| - |價格變化%|

範例：
第一波高點：股價 \$100，情緒 85
第二波高點：股價 \$110，情緒 80

價格變化% = (110 - 100) / 100 = +10%
情緒變化% = (80 - 85) / 85 = -5.9%

背離強度 = |-5.9%| - |+10%| = -4.1%（負值 = 頂背離）

```

**背離強度解讀：**

| 背離強度 | 背離程度 | 反轉機率 |
|---------|---------|---------|
| < -10% | 極強頂背離 | 80% |
| -5% ~ -10% | 強頂背離 | 65% |
| -2% ~ -5% | 中度頂背離 | 50% |
| +2% ~ +5% | 中度底背離 | 50% |
| +5% ~ +10% | 強底背離 | 65% |
| > +10% | 極強底背離 | 80% |

## 情緒轉折點預測

### 基於歷史模式的預測

**步驟 1：識別歷史相似情境**
- 當前情緒分數：85（極度樂觀）
- 搜尋歷史上情緒分數達到 85 的時刻
- 記錄後續走勢

**步驟 2：統計歷史反轉時間**
```

歷史案例分析：

案例 1（2021/1）：情緒達 88 → 2 週後見頂 → 下跌 -15%
案例 2（2021/5）：情緒達 90 → 1 週後見頂 → 下跌 -20%
案例 3（2020/8）：情緒達 85 → 3 週後見頂 → 下跌 -10%

平均反轉時間：(2 + 1 + 3) / 3 = 2 週
平均跌幅：(-15% - 20% - 10%) / 3 = -15%

```

**步驟 3：預測當前情境**
```

當前情緒分數：85
歷史相似情境平均反轉時間：2 週
預估反轉時間範圍：1-3 週內

反轉機率：75%（基於 3 個歷史案例中 3 個都反轉）
預估最大跌幅：-15%

```

---

### 轉折點觸發條件

**頂部轉折點（需同時滿足 4 項以上）：**
- ✅ 情緒分數 > 85（極度樂觀）
- ✅ 歷史分位數 > 90%
- ✅ 情緒動能轉負（上升放緩）
- ✅ 出現頂背離（連續 2 週）
- ✅ 散戶狂熱指標 > 5 項確認
- ✅ VIX 開始上升
- ✅ 機構開始減持（法人賣超）
- ✅ 出現負面催化劑（業績不佳、政策轉向）

**操作建議：**
- 🔴 **立即大幅減碼（50-70%）**
- 🔴 設定嚴格停利
- 🔴 增持現金

---

**底部轉折點（需同時滿足 4 項以上）：**
- ✅ 情緒分數 < 20（極度恐慌）
- ✅ 歷史分位數 < 10%
- ✅ 情緒動能轉正（下降放緩）
- ✅ 出現底背離（連續 2 週）
- ✅ 散戶恐慌指標 > 5 項確認
- ✅ VIX 開始下降
- ✅ 機構開始買進（法人買超）
- ✅ 基本面未惡化

**操作建議：**
- 🟢 **謹慎逢低佈局（5-10% 倉位）**
- 🟢 等待情緒穩定後加碼
- 🟢 嚴格停損（-15%）

## 情緒持續時間預測

### 情緒週期時長統計

**歷史平均週期時長（美股 S&P 500，2000-2025）：**

| 情緒階段 | 平均持續時間 | 範圍 |
|---------|------------|------|
| 極度恐慌（< 20） | 2-4 週 | 1-8 週 |
| 恐慌至懷疑（20-30） | 4-8 週 | 2-12 週 |
| 懷疑至樂觀（30-60） | 3-6 個月 | 1-12 個月 |
| 樂觀至興奮（60-75） | 2-4 個月 | 1-6 個月 |
| 興奮至狂熱（75-90） | 2-6 週 | 1-10 週 |
| 極度狂熱（> 90） | 1-3 週 | 1-6 週 |

**關鍵洞察：**
- 極端情緒（< 20 或 > 90）持續時間最短（1-4 週）
- 中性情緒（30-60）持續時間最長（3-12 個月）
- 從極度恐慌到極度狂熱的完整週期：6-18 個月

---

### 提前退出機制

**情緒持續超預期時的應對：**

**預期 1-2 週見頂，但已持續 4 週：**
1. 重新評估基本面（是否有新的支撐因素）
2. 檢視歷史最長持續時間
3. 若超過歷史最長時間，可能進入「新常態」
4. 調整預期，但仍保持警覺

**範例：**
```

2020/11-2021/11：情緒持續樂觀（60-85）長達 12 個月
原因：疫情後經濟重啟 + 低利率 + 財政刺激
→ 基本面支撐延長情緒週期

```

## 多時間框架情緒分析

### 短期 vs 中期 vs 長期情緒

**時間框架定義：**
- **短期情緒：** 近 1-2 週（即時反應）
- **中期情緒：** 近 1-3 個月（主要趨勢）
- **長期情緒：** 近 6-12 個月（結構性變化）

**多時間框架衝突解讀：**

| 短期情緒 | 中期情緒 | 長期情緒 | 市場狀態 | 操作建議 |
|---------|---------|---------|---------|---------|
| 恐慌 | 樂觀 | 樂觀 | 短線回調 | 🟢 逢低買進機會 |
| 狂熱 | 樂觀 | 樂觀 | 短線過熱 | 🟡 部分獲利，等回調 |
| 恐慌 | 恐慌 | 樂觀 | 中期調整 | 🟡 謹慎試單 |
| 恐慌 | 恐慌 | 恐慌 | 熊市 | 🔴 避險，等待長期見底 |
| 狂熱 | 狂熱 | 狂熱 | 牛市泡沫 | 🔴 大幅減碼 |
| 狂熱 | 狂熱 | 樂觀 | 牛市末期 | 🔴 獲利了結 |

**優先順序：**
- 長期情緒決定大方向（牛市 vs 熊市）
- 中期情緒決定主要趨勢
- 短期情緒決定進場時機

## 實戰操作流程

### 每週情緒預測檢查（推薦）

**步驟 1：更新情緒儀表板（每週日）**
- 記錄所有情緒指標當前值
- 計算綜合情緒分數
- 計算歷史分位數

**步驟 2：計算情緒動能與加速度**
```

當前情緒分數：78
上週情緒分數：72
上上週情緒分數：68

動能 = 78 - 72 = +6
上週動能 = 72 - 68 = +4
加速度 = +6 - (+4) = +2（正值，加速上升）

解讀：情緒加速上升，接近過熱

```

**步驟 3：檢查背離**
- 對比股價高點與情緒高點
- 若出現背離，標記警報

**步驟 4：預測轉折點**
- 查詢歷史相似情境
- 估算反轉時間與幅度

**步驟 5：操作決策**
- 根據預測調整倉位
- 設定停利/停損

**時間成本：** 約 20-30 分鐘/週

## 常見誤區

**誤區 1：情緒預測 = 100% 準確**
- 預測只是提高勝率，非絕對
- 歷史成功率約 50-70%

**誤區 2：單一指標預測**
- 需多指標交叉驗證
- 至少 4-5 項指標確認

**誤區 3：忽略基本面變化**
- 基本面改善可延長情緒週期
- 需動態調整預測

**誤區 4：過度依賴歷史模式**
- 「這次不一樣」有時是對的
- 歷史是參考，非絕對

**誤區 5：背離一出現就操作**
- 需連續 2 週以上確認
- 背離可能持續數週

## 總結

**情緒預測核心要點：**

1. **歷史分位數：** < 10% 見底，> 90% 見頂
2. **情緒動能與加速度：** 動能正但加速度負 = 可能見頂
3. **情緒-價格背離：** 連續 2 週背離 = 轉折信號
4. **轉折點觸發：** 需 4-5 項指標同時確認
5. **時間預測：** 極端情緒持續 1-4 週，中性情緒持續 3-12 個月

**實戰檢查清單：**
- ✅ 建立情緒預測儀表板
- ✅ 每週計算情緒分數與歷史分位數
- ✅ 追蹤情緒動能與加速度
- ✅ 監測情緒-價格背離
- ✅ 記錄歷史極端值時刻
- ✅ 查詢歷史相似情境
- ✅ 估算反轉時間與幅度
- ✅ 設定警報閾值

**下一步行動：**

- 每週執行情緒預測流程
