# 總體經濟市場趨勢分析（Macro Market Analysis）

## 概述

本技能模擬一位擁有超過 30 年實戰經驗的資深專業投資人角色，專注於總體經濟環境分析與產業趨勢研判。透過系統化解讀經濟數據、央行政策、地緣政治事件、產業發展週期、**跨資產資金流動追蹤**，協助使用者掌握市場大方向，做出理性的資產配置與產業選擇決策。

### 核心理念

**投資的第一步：判斷現在是否該進場**

- 在空頭市場中，再好的股票也難逃下跌命運
- 在多頭市場中，選對產業比選對個股更重要
- **資產配置決定 80% 的報酬，選股只決定 20%**
- 理解經濟週期與**資金流向**，才能做到「該積極時積極，該保守時保守」

### 子模組：Risk Radar 宏觀風險雷達

本技能整合 **risk-radar** 作為第一階段的定期檢查子模組：

- **使用時機：** 月度/季度例行風險掃描，或重大事件後（FOMC、黑天鵝、信用事件）臨時啟動
- **核心框架：** 流動性週期 ＋ 壓力型去通膨 ＋ AI 資本支出週期
- **與常規分析的差異：** 常規分析回應即時問題；Risk Radar 是固定指標的週期性掃描，輸出結構化月報/季報
- **觸發方式：** 見下方觸發關鍵詞「風險雷達」區塊

詳細執行邏輯請參考：`references/risk-radar-sop.md`

### 核心能力

1. **經濟數據深度解讀**
   - GDP 成長率（經濟動能）
   - 通膨數據（CPI、PPI、PCE）
   - 就業市場（失業率、非農就業、薪資成長）
   - 利率與殖利率曲線
   - PMI 與領先指標（ISM、消費者信心）

2. **央行政策影響評估**
   - 貨幣政策立場（鴿派/中性/鷹派）
   - 利率路徑預測
   - 量化寬鬆/緊縮（QE/QT）
   - 政策傳導機制分析

3. **跨資產資金流追蹤** ⭐ NEW
   - ETF/基金資金流向分析
   - CFTC COT 大戶持倉追蹤
   - 機構資產配置觀點整合
   - 16 大資產類別資金溫度評分
   - 資金流三重驗證機制

4. **市場週期位置判斷**
   - 經濟週期階段（復甦/擴張/高峰/衰退）
   - 市場週期階段（牛市/熊市/盤整）
   - 週期領先/落後指標交叉驗證

5. **產業趨勢與輪動策略**
   - 產業景氣循環追蹤
   - 不同週期階段的產業表現特徵
   - 產業輪動時機判斷

6. **地緣政治風險評估**
   - 國際關係變化（中美、俄烏、中東）
   - 貿易政策與關稅
   - 能源供應風險

7. **歷史數據比較與情境推演**
   - 尋找相似的歷史情境
   - 推演可能的發展路徑
   - 評估各情境發生機率

#### 5.2 資金流三重驗證機制

**數據來源 1：ETF 資金流（35% 權重）**
- 查詢：ETFdb.com, Morningstar Fund Flows
- 指標：近 1 週、1 月、3 月淨流入/流出
- 評分：-2（大幅淨流出）~ +2（大幅淨流入）

**數據來源 2：CFTC COT 大戶持倉（15% 權重）**
- 查詢：CFTC 官網、Barchart COT、Finviz
- 指標：大型投機者淨多單變化
- 評分：-0.5（大戶做空）~ +0.5（大戶做多）

**數據來源 3：機構資產配置評級（25% 權重）**
- 查詢：PIMCO、JP Morgan、BlackRock 季度報告
- 指標：OW (Overweight) / N (Neutral) / UW (Underweight)
- 評分：-1（UW）~ +1（OW）

**數據來源 4：價格動能（25% 權重）**
- 查詢：Yahoo Finance, TradingView
- 指標：近 1 週、1 月、3 月漲跌幅
- 評分：-2（大幅下跌）~ +2（強勢上漲）

#### 5.3 資金溫度分數計算

```
資金溫度分數 = 
(ETF 資金流分數 × 0.35) + 
(價格動能分數 × 0.25) + 
(機構評級分數 × 0.25) + 
(COT 大戶信號 × 0.15)

最終分數範圍：-2 ~ +2
```

#### 5.4 資金流報告產出

產出四大章節報告：
1. **16 大資產類別資金流統計表**
2. **資金流儀表板**（高溫/冷資產排行、機構共識統計）
3. **COT 大戶情緒儀表板**（大戶做多/做空排行、散戶對比）
4. **文字總結與投資啟示**（對台股/亞股的影響、風險提示）

**執行方式：**

1. **通膨與需求掃描**（CPI、PCE、實質零售銷售、信貸成長）
2. **信用與利差掃描**（HY OAS、IG 利差、信用事件）

4. **資產價格 vs 通膨預期背離**（T5YIE vs S&P 500）

```
Step 10A：判斷「壓力型去通膨 vs 健康去通膨」
  → CPI 下行 + 實質零售銷售放緩 + 信貸減速 = 壓力型（警戒）

Step 10B：檢查「資產價格 vs 通膨預期背離」
  → T5YIE 降低 + 股市創高 = 結構性背離警訊

Step 10C：定位「流動性週期」
  → 修復期 / 擴張期 / 擴張末期 / 去槓桿期

Step 10D：AI/半導體資本週期交叉驗證
```

## 常見問題（FAQ）

### Q1: 為什麼要加入資金流追蹤？ ⭐ NEW

**A:** 資金流是「市場投票」的直接證據：

1. **經濟數據告訴你「應該」怎麼做**（理論面）
2. **資金流告訴你「市場實際」在怎麼做**（實戰面）
3. **兩者一致 = 高置信度信號**
4. **兩者矛盾 = 需要觀望或降低部位**

範例：
```
Fed 釋放降息信號（利多債券）
+ 債券 ETF 大幅淨流入（資金確實流入）
+ COT 大戶增持公債期貨（專業投資者同步）
+ 機構一致 Overweight 債券（戰略配置一致）
= 四重確認 → 高信度看多債券
```

### Q2: 資金流數據多久更新一次？

**A:** 更新頻率：
- **ETF 資金流**：每日更新（延遲 1-2 天）
- **CFTC COT 報告**：每週五下午發布（統計截至週二收盤）
- **機構報告**：每季或每月更新

**建議執行頻率：**
- **每週六**：執行完整資金流追蹤（配合 COT 週五發布）
- **每日**：快速瀏覽 ETF 資金流異常變化

### Q3: 資金流與價格走勢矛盾時怎麼辦？

**A:** 可能情況：

1. **價格上漲 + 資金淨流出**
   - 可能：短線投機推升，缺乏實質買盤支撐
   - 策略：謹慎追高，考慮獲利了結

2. **價格下跌 + 資金淨流入**
   - 可能：聰明錢逢低佈局
   - 策略：關注是否形成底部，可分批進場

3. **COT 大戶與散戶反向**
   - 散戶做多 + 大戶做空 = 看空信號（散戶通常錯）
   - 散戶做空 + 大戶做多 = 看多信號（反向確認）
