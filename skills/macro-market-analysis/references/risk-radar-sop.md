## 三、四步驟分析邏輯

### Step 1：判斷「壓力型去通膨 vs 健康去通膨」

```
壓力型去通膨（警戒）= 以下三者同時出現：
  ✓ CPI / PCE 明確下行
  ✓ 實質零售銷售放緩或轉負
  ✓ 信貸成長減速

判斷後需回答：
  → 通膨下行是需求變弱？還是供給改善？
  → 對企業 EPS 的壓力何時顯現？
```

### Step 2：檢查「資產價格 vs 通膨預期」背離

```
背離警訊 = T5YIE 高點逐波降低 + S&P 500 維持高位或創新高
  → 標記為「結構性背離警訊」
  → 通常為危機前 6-18 個月的型態
  → 建議：降低新增風險資產投入、鎖定部分獲利
```

### Step 3：定位「流動性週期」

```
修復/早期擴張期：
  → 多數指標改善 + 金融條件寬鬆 + 曲線正斜率

擴張末期/壓力累積期：
  → 增速放緩 + 曲線長期倒掛 + 金融條件開始收緊

去槓桿期：
  → 利差急擴 + 信用事件頻出 + 金融條件急速收緊
```

### Step 4：AI/半導體資本週期交叉驗證

```
AI capex 高檔 + 營收成長放緩 + 毛利率受壓
  → AI 投資與現金流回收錯位，可能放大宏觀壓力

AI 需求強勁 + 順利轉為營收與利潤
  → 對風險資產相對友善，但仍受宏觀流動性約束
```
