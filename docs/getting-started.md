## 📱 裝置使用指南

### 手機使用 (iOS / Android)

1. **長按連結** → 選擇「複製」
2. **開啟 AI App** (Perplexity / Claude / ChatGPT)
3. **長按對話框** → 選擇「貼上」
4. **加上你的問題** → 傳送!

### 電腦使用 (Windows / Mac)

1. **選取連結** → `Ctrl+C` / `Cmd+C`
2. **開啟 AI 網站**
3. **對話框中** → `Ctrl+V` / `Cmd+V`
4. **加上你的問題** → Enter!

## ❓ 常見問題

### Q: AI 說無法讀取連結?

{: .warning }
某些 AI 可能需要明確指令。試試:

```

請存取並完整讀取以下 GitHub 檔案:
[貼上連結]

讀取完成後,請確認你已理解框架內容。

```

如果還是不行:
1. 點擊連結 → 全選內容 → 複製
2. 直接貼給 AI
3. 或使用 Perplexity / Claude (支援較好)

---

### Q: 需要每次都貼連結嗎?

{: .note }
**同一對話串**: 不需要,AI 會記住  
**新對話串**: 需要重新載入

**省時技巧:**
- Claude: 使用「Projects」功能
- ChatGPT: 設定「Custom Instructions」
- Perplexity: 在設定中新增「Context」

---

### Q: 免費版 AI 夠用嗎?

{: .highlight }
免費版完全夠用!但付費版體驗更好:

| 平台 | 免費版 | 付費版 ($20/月) |
|-----|-------|---------------|
| Perplexity | ✅ 可用 | 無限 Pro 搜尋 |
| Claude | ✅ 可用 | 更長對話深度 |
| ChatGPT | ✅ 可用 | 完整功能 |

**建議:** 先用免費版試試,有需要再升級。

---

### Q: 如何驗證 AI 的分析?

{: .important }
**務必交叉驗證!**

**數據來源:**
- 📊 財務數據: 對照公司財報、證交所
- 📰 新聞資訊: 查證原始來源
- 📈 股價資料: 對照券商平台

**推薦驗證平台:**
- **美股**: SEC EDGAR, Yahoo Finance
- **台股**: 公開資訊觀測站, CMoney
- **港股**: 港交所披露易
- **陸股**: 巨潮資訊網

## 🎓 進階使用

### 技巧 1: 分階段深入

```

我想對 Apple (AAPL) 進行完整分析:

第一步 - 載入框架:
https://raw.githubusercontent.com/.../FRAMEWORK.md

第二步 - 逐步執行:
請先完成階段一(總體經濟),用最新數據。
完成後我再請你繼續。

```

### 技巧 2: 比較多支股票

```

請用此框架比較:

- Apple (AAPL)
- Microsoft (MSFT)
- Google (GOOGL)

框架: https://raw.githubusercontent.com/.../SKILL.md

產出比較表,包含: 基本面評分、財務健康度、競爭優勢

```

### 技巧 3: 建立觀察清單

```

請用框架篩選科技股中最值得投資的前 5 家:

產業框架: [連結 1]
個股框架: [連結 2]

標準: 產業領先、財務健康、成長性佳、估值合理

```
